# AI 萌新小窝 - 项目说明文档

## 📖 项目简介

**AI 萌新小窝**是一个专门为完全不懂编程和AI的新人打造的科普网站，旨在通过简单易懂的方式，帮助新人了解AI的基本概念、术语、工具使用，以及如何利用AI进行文学创作。

### 核心目标
- **科普AI基础知识**：用通俗易懂的语言解释AI概念和术语
- **分享实用工具**：推荐适合新手的AI工具和使用方法
- **提供创作指导**：教新人如何用AI进行文学创作
- **传递科技资讯**：分享最新的AI科技新闻和热门项目

## 🚀 网站功能

### 1. 首页
- **热搜排行**：展示本周热门AI话题
- **精选文章**：分享优质AI技术文章和资源
- **搞钱排行**：介绍AI相关的赚钱机会
- **AI创作**：展示AI创意作品和创作技巧
- **萌新学习**：提供AI入门学习资料
- **实战案例**：分享AI应用的实际案例

### 2. 资源导航
- 精选AI工具和资源推荐
- 分类整理，方便新手查找

### 3. AI学习
- 系统化的AI学习路径
- 从基础到进阶的学习资源

### 4. 游戏开发
- AI在游戏开发中的应用
- 游戏相关的AI工具和技术

### 5. 经典对话
- AI领域专家观点碰撞
- 深度讨论AI发展趋势

### 6. 创作者交流
- 创作者交流互动平台
- 作品分享、问题咨询、经验交流
- 支持类型分类：问题咨询、作品分享、经验交流、建议反馈

## 🎯 目标用户

- **完全新手**：对AI和编程一无所知的人
- **职场人士**：希望利用AI提升工作效率的人
- **学生**：想了解AI知识的学习者
- **创作者**：希望用AI辅助创作的人
- **好奇人士**：对AI技术感兴趣的普通大众

## 📚 内容体系

### AI基础知识
- **概念科普**：AI、机器学习、深度学习等基本概念
- **术语解释**：Prompt、Token、大模型等常见术语
- **发展历程**：AI技术的发展历史和重要里程碑

### 工具使用指南
- **聊天工具**：ChatGPT、Claude、豆包、通义千问等
- **创作工具**：AI写作、AI绘画、AI音乐等工具
- **效率工具**：AI辅助办公、学习、生活的工具

### 文学创作指导
- **小说创作**：用AI写小说的方法和技巧
- **剧本创作**：AI辅助剧本写作
- **诗词创作**：用AI写诗作词
- **社交媒体文案**：AI辅助写朋友圈、微博等内容
- **幽默段子**：用AI写搞笑内容

### 科技资讯
- **最新动态**：AI领域的最新新闻和发展
- **热门项目**：GitHub上的热门AI项目
- **行业应用**：AI在各行业的应用案例

## 🛠️ 技术架构

### 前端技术
- **HTML5**：页面结构
- **CSS3**：样式设计，包括响应式布局
- **JavaScript**：交互功能和内容渲染
- **Markdown**：内容管理和渲染

### 内容管理
- **Skill系统**：用于生成各种类型的内容
- **数据目录**：按分类存储Markdown格式的内容
- **前端渲染**：动态加载和展示内容

### 响应式设计
- 适配桌面端、平板和移动设备
- 移动端优化的导航和布局
