## 📖 创作指南

### 第一步：准备工作

**明确文案类型**
- 朋友圈：个人生活分享、情感表达
- 小红书：产品推荐、攻略教程、种草文案
- 微博：热点评论、话题互动、短内容

**确定目标受众**
- 朋友圈：亲朋好友，语气亲切自然
- 小红书：年轻女性，语气活泼种草
- 微博：大众用户，语气观点鲜明

**收集素材**
- 照片/视频内容
- 产品特点或使用体验
- 想表达的核心观点

### 第二步：Prompt 设计

**朋友圈文案模板：**
```
请帮我写一条朋友圈文案，内容是[描述内容]。
要求：
1. 语气[亲切/幽默/文艺/励志]
2. 字数控制在[50-100]字
3. 适合配[照片/视频/纯文字]
```

**小红书文案模板：**
```
请帮我写一篇小红书种草文案，推荐[产品名称]。
要求：
1. 标题要吸引人点击
2. 内容包含使用体验、优点、适合人群
3. 语气活泼亲切，多用emoji
4. 包含相关话题标签
5. 字数300-500字
```

**示例 Prompt：**
```
请帮我写一篇小红书文案，推荐一款保温杯。
要求：
1. 标题要吸引人点击
2. 内容包含保温效果、外观设计、使用感受
3. 语气活泼亲切，多用emoji
4. 包含#保温杯推荐 #好物分享 等话题标签
5. 字数400字左右
```

### 第三步：内容生成

**生成初稿**
1. 输入Prompt，让AI生成多个版本
2. 选择最符合你需求的版本
3. 对比不同版本的优缺点

**人工编辑和优化**
1. **标题优化**：确保标题足够吸引人
2. **内容调整**：加入个人真实体验
3. **语气调整**：让文案更符合你的风格
4. **格式优化**：合理分段，适当添加emoji

**最终成品**
- 检查错别字和标点
- 确保信息准确（价格、规格等）
- 配上合适的图片或视频

## ⚠️ 注意事项

### 注意 1：真实性问题
**问题**：AI可能编造虚假的产品信息  
**解决**：所有产品信息必须你自己核实，AI只负责文案表达

### 注意 2：同质化风险
**问题**：AI生成的文案可能和其他人相似  
**解决**：加入个人真实体验和独特观点，让文案有个人特色

### 注意 3：平台规则
**问题**：不同平台有各自的文案风格和规范  
**解决**：在Prompt中明确平台要求，生成后还要符合平台规则

## 💡 创作技巧

### 标题技巧
- **数字法**："3个让我离不开的保温杯理由"
- **疑问法**："你还在用普通保温杯？"
- **对比法**："百元vs千元保温杯，差距在哪？"
- **情感法**："冬天喝上一口热水的幸福"

### 内容技巧
- **个人体验**：AI写的再好看，不如一句"我用了三个月的真实感受"
- **场景代入**：描述具体使用场景，让读者有画面感
- **痛点解决**：先说用户痛点，再说产品如何解决
- **行动号召**：结尾引导读者互动，如"你们有什么推荐？"
