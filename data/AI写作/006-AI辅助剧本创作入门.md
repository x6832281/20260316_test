## 🔍 什么是AI辅助剧本创作？

AI辅助剧本创作是利用AI工具帮你写剧本的方法。无论是短视频剧本、微电影、话剧剧本、小品，AI都能帮你构思情节、设计对话、安排场景，让创意变成可拍摄的剧本。

## 📖 创作指南

### 第一步：准备工作

**确定剧本类型**
- 短视频剧本：1-3分钟，节奏快，反转多
- 微电影剧本：5-15分钟，故事完整
- 小品剧本：喜剧为主，适合舞台表演
- 话剧剧本：多幕剧，人物对话为主

**明确核心创意**
- 你想讲一个什么故事？
- 核心冲突是什么？
- 想传达什么主题？

**收集素材**
- 生活观察和体验
- 热门话题和社会现象
- 经典剧本的结构学习

### 第二步：Prompt 设计

**短视频剧本模板：**
```
请帮我写一个短视频剧本，类型是[类型]。
要求：
1. 时长[1-3]分钟
2. 场景[数量]个
3. 人物[数量和特征]
4. 必须有反转/笑点/泪点
5. 格式：场景描述 + 对话
```

**示例 Prompt：**
```
请帮我写一个搞笑短视频剧本。
要求：
1. 时长2分钟左右
2. 场景2个：办公室和电梯
3. 人物2个：老板和员工
4. 必须有反转，结局出人意料
5. 格式：场景描述 + 对话
```

**微电影剧本模板：**
```
请帮我写一个微电影剧本大纲。
主题：[主题描述]
要求：
1. 时长10分钟
2. 三幕结构：开端、发展、高潮、结尾
3. 人物3-4个
4. 包含场景描述、对话、动作提示
5. 情感[感人/励志/悬疑]
```

### 第三步：内容生成

**生成初稿**
1. 输入Prompt，生成剧本初稿
2. 检查结构是否完整
3. 评估节奏和冲突

**人工编辑和优化**
1. **节奏调整**：短视频要快，微电影要有起伏
2. **对话优化**：让对话更自然、口语化
3. **冲突强化**：增加戏剧张力
4. **场景精简**：去除不必要场景

**最终成品**
- 按格式排版（场景、人物、对话）
- 标注拍摄要点
- 准备分镜脚本

## ⚠️ 注意事项

### 注意 1：格式规范
**问题**：AI生成的剧本格式可能不规范  
**解决**：学习标准剧本格式，人工调整

### 注意 2：可操作性
**问题**：AI可能写出无法拍摄的场景  
**解决**：考虑实际拍摄条件，调整场景设计

### 注意 3：节奏把控
**问题**：AI可能节奏拖沓或过快  
**解决**：人工调整节奏，确保符合时长要求

## 💡 创作技巧

### 短视频剧本技巧
- **黄金3秒**：开头3秒必须抓住观众
- **反转设计**：结尾反转，出乎意料
- **情绪共鸣**：让观众有代入感
- **简洁对白**：短视频对话要精炼

### 结构技巧
- **三幕结构**：
  - 第一幕（25%）：介绍人物和冲突
  - 第二幕（50%）：冲突升级
  - 第三幕（25%）：高潮和解决
- **起承转合**：
  - 起：引入
  - 承：发展
  - 转：转折
  - 合：结尾

### 对话技巧
- **符合人物**：不同角色说话方式不同
- **推动情节**：对话要有目的性
- **潜台词**：不要把所有话说透
- **动作配合**：对话配合动作描写

## 📝 实战案例

### 案例：用AI写搞笑短视频剧本

**需求**：写一个办公室搞笑短视频剧本

**步骤**：
1. **设定要求**：2分钟、办公室和电梯场景、老板和员工、反转结局
2. **AI生成**：生成初稿
3. **人工修改**：
   - 优化对话，更口语化
   - 调整节奏，加快开头
   - 强化反转效果

**最终剧本**：

**《加班的秘密》**

**场景1：办公室，晚上9点**

老板（走过员工工位）：你怎么还在？

员工（盯着电脑）：加班赶方案。

老板（满意点头）：好，好好干。

（老板离开）

员工（拿起手机打游戏）：终于走了，继续上分。

**场景2：电梯，晚上9点半**

（员工进电梯，老板在里面）

员工（慌张）：老板，您还没走啊？

老板（微笑）：等你啊，就想看看你是不是真在加班。

员工（尴尬）：那个...我在...

老板（拿出手机）：别解释了，你游戏ID多少？我等你组队。

员工（震惊）：？？？

老板：其实我每天晚上都在公司打游戏，就是缺个队友。

**【反转结局】**
