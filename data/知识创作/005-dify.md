# Dify —— LLM 应用开发与知识创作平台

**⭐ 星标**：52.0k | **📅 更新日期**：2026-04-27

---

## 📝 一句话总结

国产开源的 LLM 应用开发平台，可视化构建 AI 知识库、写作助手和内容生成流水线。

## 📋 详细介绍

Dify 是国产最优秀的 LLM 应用开发平台之一，提供了可视化的工作流编排能力，让不懂代码的创作者也能构建自己的 AI 写作应用：

- 知识库管理：上传文档、网页、Notion 笔记，构建专属写作素材库
- 可视化工作流：拖拽式编排搜集→分析→写作→润色→发布的完整流程
- 提示词工程：可视化管理提示词模板，支持变量和条件逻辑
- AI Agent：配置能够自主搜索、阅读、写作的智能体
- 日志与标注：记录每次 AI 调用的输入输出，持续优化写作质量
- 多模型支持：接入 OpenAI、Claude、通义千问、DeepSeek 等 100+ 模型

Dify 特别适合需要批量创作内容、建立标准化写作流程的团队使用。

## 🎯 个人评价

优点：可视化编排、知识库强大、国产友好、社区活跃。缺点：自部署需要一定技术基础。适合人群：内容团队、自媒体人、知识管理爱好者。

## 🔧 技术栈

- Python / Flask
- React / Next.js
- PostgreSQL
- Redis / Celery
