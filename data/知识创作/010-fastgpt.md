# FastGPT —— AI 知识库问答与内容生成平台

**⭐ 星标**：22.0k | **📅 更新日期**：2026-04-27

---

## 📝 一句话总结

国产开源AI知识库平台，拖拽式构建写作助手，支持RAG检索增强生成，让AI基于你的素材写作。

## 📋 详细介绍

FastGPT 是国产最优秀的 AI 知识库平台之一，为知识创作者提供了开箱即用的内容生成和管理能力：

- 知识库工作流：可视化拖拽编排数据导入 → 文本预处理 → 向量化 → 问答匹配全流程
- RAG 增强写作：上传你的参考素材，AI 基于你的专属知识库进行创作
- 多格式导入：支持 PDF、Word、网页、CSV、手动输入等多种导入方式
- 内容生成模板：预设写作、翻译、总结、润色等工作流模板
- API 开放：提供完整 API，可嵌入到自己的网站或应用中
- 团队协作：支持多人共享知识库和写作工作流

相比通用 AI 对话，FastGPT 的优势在于让你的 AI 真正"读懂"你的素材，写出更准确、更相关的内容。

## 🎯 个人评价

优点：知识库功能强大，可视化编排，国产友好，部署简单。缺点：对硬件有一定要求，高级功能需要配置。适合人群：内容团队、知识管理需求者、自媒体创作者。

## 🔧 技术栈

- TypeScript / Next.js
- MongoDB / PostgreSQL
- 向量数据库
- Docker 部署
