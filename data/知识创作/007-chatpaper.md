# ChatPaper —— AI 论文阅读与翻译工具

## 📝 一句话总结

arXiv论文一键翻译+摘要，AI辅助文献综述，科研写作者必备的论文阅读加速器。

## 📋 详细介绍

ChatPaper 是一款专为学术写作和文献研究设计的开源工具。它能够自动获取 arXiv 最新论文，用 AI 生成中文摘要、全文翻译，并支持论文对话问答：

- 全流程自动化：从 ArXiv 检索 → 下载 → AI 精读 → 生成摘要，一键完成
- 论文对话：上传 PDF，像聊天一样向 AI 提问论文中的任何细节
- 多模型支持：可接入 GPT-4、Claude、DeepSeek 等多种大模型
- 文献综述辅助：批量分析多篇论文，自动提取共同主题和争议点
- 笔记导出：AI 生成的摘要和笔记可导出为 Markdown/PDF

对于需要进行大量文献阅读和综述写作的研究生、科研人员来说，这是效率提升神器。

## 🎯 个人评价

优点：论文翻译质量高，全流程自动化，开源免费。缺点：依赖大模型 API 费用，部分论文排版解析有瑕疵。适合人群：研究生、科研人员、学术写作者。

## 🔧 技术栈

- Python
- ArXiv API
- LLM API 集成
- PDF 解析
