## 🎯 为什么要做内容矩阵？

写一篇文章要2小时，但只发一个平台太亏了。同样的内容，稍微改写就能适配5个平台，覆盖5倍读者。

但手动改写太慢——每个平台的风格、长度、格式都不同。AI可以帮你把改写时间从2小时缩短到20分钟。

## 📊 5大平台内容特征对比

| 平台 | 字数 | 风格 | 标题 | 开头 | 结尾 |
|------|------|------|------|------|------|
| 知乎 | 2000-5000 | 专业、有逻辑 | 提问式 | 先给结论 | 总结+延伸阅读 |
| 小红书 | 300-800 | 口语、有情绪 | 数字+emoji | 痛点共鸣 | 互动引导 |
| 公众号 | 1500-3000 | 深度、有温度 | 悬念式 | 故事引入 | 金句收尾 |
| 抖音口播 | 200-400 | 短平快、有节奏 | 悬念钩子 | 3秒抓人 | 引导关注 |
| 微博 | 100-200 | 金句、有观点 | 话题标签 | 直接亮观点 | 转发引导 |

## ✍️ 实战：一篇文章改写5个版本

假设你写了一篇拆书笔记：《活着·福贵买牛：极简叙事的终极力量》

### 版本1：知乎长文

```
改写Prompt：
请将以下拆书笔记改写为知乎回答格式，要求：
1. 字数2000-3000字
2. 先给结论，再展开分析
3. 引用原文时用引用格式
4. 每个分析点用小标题分隔
5. 结尾加"延伸阅读"推荐

[粘贴原文]
```

### 版本2：小红书短文

```
改写Prompt：
请将以下拆书笔记改写为小红书笔记格式，要求：
1. 字数300-500字
2. 标题用数字+emoji（如"3个写作技法🔥"）
3. 开头写痛点（"你是不是也……"）
4. 正文用短句+emoji分隔
5. 结尾加互动引导（"你最想拆哪本书？评论区告诉我👇"）

[粘贴原文]
```

### 版本3：公众号中篇

```
改写Prompt：
请将以下拆书笔记改写为公众号文章格式，要求：
1. 字数1500-2000字
2. 标题用悬念式（如"余华最狠的一招，99%的人没看出来"）
3. 开头用一个小故事引入
4. 正文有节奏感：分析→案例→洞察→练习
5. 结尾用金句收尾

[粘贴原文]
```

### 版本4：抖音口播脚本

```
改写Prompt：
请将以下拆书笔记改写为抖音口播脚本，要求：
1. 字数200-300字（约1分钟口播）
2. 开头3秒抓人（"你知道余华最狠的写法是什么吗？"）
3. 全部用口语，短句为主
4. 不用任何书面语和成语
5. 结尾引导关注（"关注我，每天拆一本好书"）

[粘贴原文]
```

### 版本5：微博金句

```
改写Prompt：
请从以下拆书笔记中提取5条金句，每条不超过50字，适合微博传播：
1. 每条金句独立成句，不需要上下文
2. 用反常识或反转制造冲击力
3. 加1-2个话题标签

[粘贴原文]
```

## 💡 矩阵运营3个技巧

### 1. 发布时间错开
- 知乎：工作日午休（12:00-13:00）
- 小红书：晚上8-10点
- 公众号：早上7-8点
- 抖音：晚上7-9点
- 微博：中午12点、晚上8点

### 2. 跨平台引流
- 知乎回答末尾："更多拆书笔记，关注我的公众号XXX"
- 小红书笔记末尾："完整版在我的公众号XXX"
- 抖音视频末尾："文字版在知乎搜XXX"

### 3. 数据追踪
用Excel或Notion记录每篇文章在各平台的数据（阅读量、点赞、评论、转发），每周复盘，找出哪个平台、哪种内容效果最好。
