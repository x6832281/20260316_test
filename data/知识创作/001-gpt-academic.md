# gpt_academic —— AI 学术写作助手

## 📝 一句话总结

中科院出品的高颜值AI学术写作助手，支持论文润色、翻译、代码解读、PDF分析等一站式学术写作功能。

## 📋 详细介绍

gpt_academic 是由中国科学院自动化研究所团队开发的开源项目，专为学术研究和知识创作设计。它将 GPT 等大语言模型的能力与学术写作场景深度结合，提供论文润色、中英互译、代码解读、PDF 论文分析、Latex 论文一键生成等强大功能。

项目支持接入 GPT-4、Claude、DeepSeek、Qwen 等多种大模型，并提供美观的 Web 界面。特别适合研究生、科研人员和需要进行深度知识创作的用户。

亮点功能包括：一键论文润色（支持多轮迭代）、Arxiv 论文翻译与摘要、项目代码自动分析、Markdown 格式报告生成等。

## 🎯 个人评价

优点：学术场景深度优化，功能全面，界面美观，更新活跃。缺点：配置稍复杂，需要 API Key。适合人群：研究生、科研人员、学术写作者。

## 🔧 技术栈

- Python
- Gradio
- GPT/Claude API
- Markdown
