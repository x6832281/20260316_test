## 🚀 快速上手（5 分钟搞定）

### 第一步：安装部署

```bash
# 克隆项目
git clone https://github.com/binary-husky/gpt_academic.git
cd gpt_academic

# 安装依赖
pip install -r requirements.txt

# 配置 API Key（编辑 config.py）
# 找到 API_KEY 字段，填入你的 OpenAI/Claude/DeepSeek API Key
```

### 第二步：启动服务

```bash
python main.py
```

浏览器打开 `http://localhost:7860` 即可使用。

### 第三步：配置模型

推荐配置：
- **日常使用**：DeepSeek（便宜、中文好）
- **论文润色**：GPT-4 或 Claude（质量高）
- **代码分析**：Claude（代码理解能力强）

## 📚 核心功能详解

### 1. 论文润色（最常用）

**使用场景**：论文写完后，需要提升英文表达质量。

**操作步骤**：
1. 在左侧文本框粘贴你的论文段落
2. 点击"润色"按钮
3. 查看修改建议，可选择"接受"或"继续润色"

**实用技巧**：
- 一次只润色一个段落，效果更好
- 先用中文写完逻辑，再让 AI 翻译+润色，比直接写英文快 3 倍
- 多轮润色时，告诉 AI "保持原意，只改表达"，避免改变你的观点

### 2. 论文翻译

**使用场景**：需要快速理解英文论文，或把中文论文翻译成英文。

**操作步骤**：
1. 点击"翻译"功能
2. 粘贴需要翻译的内容
3. 选择源语言和目标语言

**实用技巧**：
- 翻译整篇论文时，按章节分别翻译，保持术语一致性
- 翻译后自己检查专业术语，AI 有时会翻错领域术语

### 3. PDF 论文解析

**使用场景**：快速了解一篇论文的核心内容，不用从头读到尾。

**操作步骤**：
1. 上传 PDF 文件
2. 点击"解析"
3. AI 会自动提取：研究目的、方法、结果、结论

**实用技巧**：
- 先看 AI 摘要，再决定是否要精读全文
- 对于重要论文，可以让 AI "用中文详细解释每个图表"

### 4. 代码解读

**使用场景**：看不懂论文附带的代码，或 GitHub 项目的代码。

**操作步骤**：
1. 粘贴代码或上传代码文件
2. 点击"代码解读"
3. AI 会逐行解释代码逻辑

## 💡 进阶使用技巧

### 技巧一：自定义快捷按钮

GPT Academic 支持自定义"快捷按钮"，把常用的操作封装成一键功能：

```python
# 在 config.py 中添加自定义按钮
CUSTOM_BUTTONS = {
    "论文降重": "请在不改变原意的前提下，用不同的表达方式重写以下内容：",
    "摘要生成": "请根据以下论文内容，生成一段 200 字的中文学术摘要：",
    "参考文献格式": "请将以下参考文献转换为 APA 格式："
}
```

### 技巧二：多模型对比

同时调用多个模型，对比哪个润色效果更好：

1. 在设置中开启"多模型对比"模式
2. 同一段落会分别用 GPT-4、Claude、DeepSeek 处理
3. 你可以直观对比不同模型的输出质量

### 技巧三：批量处理

如果有大量文献需要翻译或摘要：

1. 准备一个文本文件，每段用 `---` 分隔
2. 使用"批量处理"功能
3. 一次性处理多个段落，节省时间

## ⚠️ 使用注意事项

### 避坑指南

1. **不要完全依赖 AI 润色**
   - AI 可能会改变你的原意，一定要自己检查
   - 特别是专业术语，AI 有时会用错

2. **注意 API 费用**
   - GPT-4 调用费用较高，建议先用 DeepSeek 测试
   - 可以在 config.py 中设置每日费用上限

3. **保护论文隐私**
   - 使用本地部署版本，不要把未发表的论文发到第三方服务
   - 如果必须用云端，先脱敏处理

4. **学术诚信**
   - AI 辅助润色是允许的，但不能让 AI 替你写论文
   - 最终内容必须是你的原创思想

## 🔧 常见问题解答

**Q：支持哪些大模型？**
A：GPT-4、Claude、DeepSeek、Qwen、GLM 等主流模型都支持，在 config.py 中配置 API Key 即可。

**Q：需要什么配置？**
A：本地部署只需要 Python 3.8+ 和 4GB 内存。不需要 GPU，因为推理是调用云端 API。

**Q：和直接用 ChatGPT 有什么区别？**
A：GPT Academic 针对学术场景做了深度优化：内置润色、翻译、摘要等专用功能，支持 PDF 解析、代码解读，界面更适合学术工作流。

**Q：免费吗？**
A：软件本身免费开源，但需要自己准备大模型的 API Key（会产生调用费用）。
