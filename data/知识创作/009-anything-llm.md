# AnythingLLM —— 全功能桌面 AI 写作伴侣

**⭐ 星标**：32.0k | **📅 更新日期**：2026-04-27**

---

## 📝 一句话总结

把任何文档变成 AI 知识库，全能型桌面 AI 应用，写作、研究、整理一站式搞定。

## 📋 详细介绍

AnythingLLM 是一个功能强大的开源桌面 AI 应用，将文档管理、知识库构建和 AI 对话深度整合：

- 文档知识库：拖入 PDF、Word、TXT、网页，自动构建写作素材库
- 多模型切换：同时配置 Ollama 本地模型和云端 API，灵活使用
- 工作空间管理：按项目/主题创建独立的工作空间，写作场景互不干扰
- 自定义 Agent：配置 AI 代理自动执行研究、总结、创作等任务
- 写作辅助：边写边问 AI，不打断创作流程
- 跨平台：支持 Windows / macOS / Linux，一键安装

特别适合需要大量参考资料创作的长文写作者、知识管理爱好者和自媒体创作者。

## 🎯 个人评价

优点：界面美观、文档管理强、支持本地+云端混合使用。缺点：初次加载时间较长，高级功能配置有学习成本。适合人群：长文写作者、研究员、知识管理爱好者。

## 🔧 技术栈

- React / Node.js
- Electron
- ChromaDB / Pinecone
- LLM API
