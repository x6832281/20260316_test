# AI 写作提示词实战：5个让文字"去AI味"的核心技巧

## 📝 一句话总结

同样的大模型，好的提示词能让输出质量差10倍——这5个技巧帮你告别AI味和流水账。

## 🎯 为什么你的AI写出来的东西"一眼AI"？

AI写作最大的问题不是"写不好"，而是"写得太好"——太流畅、太全面、太正确。真正的好文字是有呼吸的、有缺口的、有个性的。以下5个技巧，就是教你怎么让AI写出"像人写的"文字。

## 技巧一：角色设定法——给AI一个具体的人

```
❌ 差的写法："写一篇科幻小说开头"

✅ 好的写法："你是一个30岁的科幻作家，之前是物理系出身，写东西喜欢用短句，从不使用感叹号。请以第一人称视角，写一个发生在火星殖民地的悬疑故事开头，200字以内。"
```

**关键点**：不要只说"写什么"，要说"谁在写"——年龄、背景、写作习惯、禁忌，越具体越好。

## 技巧二：反例排除法——告诉AI"不要怎样"

AI最大的问题是"用力过猛"——它会把每个句子都写得华丽、每个结尾都写得升华。你需要明确告诉它不要做什么：

```
请改写以下段落，要求：
- 不要使用任何四字成语
- 不要使用感叹号
- 不要在结尾升华主题
- 不要使用"仿佛""宛如""犹如"
- 每句话不超过15个字
```

**关键点**：负面指令比正面指令更有效。告诉AI"不要怎样"，比告诉它"要怎样"更能控制输出质量。

## 技巧三：分步引导法——拆解复杂写作任务

不要一次让AI写一整篇文章，分步来：

**第1步：讨论主题**
```
我想写一篇关于"城市独居者"的散文，主题是"孤独不是一种状态，是一种能力"。请给我3个切入角度，每个角度用一句话概括。
```

**第2步：确定结构**
```
我选第二个角度。请帮我拟一个5段式的结构大纲，每段写什么、用什么素材、什么情感基调。
```

**第3步：逐段写作**
```
请写第一段，要求：用第二人称"你"，开头用一个具体的场景（不要抽象描述），200字以内。
```

**第4步：润色去AI味**
```
请检查以下段落中的AI味表达并替换：
- "宛如"→删除或改写
- "令人"→删除
- "不禁"→删除
- 任何四字成语→用口语化表达替代
```

**关键点**：每一步都给反馈，不要让AI一口气写完。你参与的越多，结果越像"你写的"。

## 技巧四：范文锚定法——给AI一个模仿对象

```
请模仿以下段落的风格，写一段描写城市清晨的文字：

参考范文：
"天还没亮，卖早点的已经把摊子支起来了。油锅里的气泡一个接一个地破，白烟直直地往上蹿，被风一扯就散了。有人蹲在路边吃油条，吃得很急，好像赶着去哪里。"

要求：
1. 用短句，每句不超过15字
2. 只写看到的，不写感受到的
3. 用"动作"代替"形容词"
4. 200字以内
```

**关键点**：给AI一个"范文"比给100条规则都管用。范文让AI知道你要的"味道"是什么。

## 技巧五：迭代打磨法——让AI自己改自己

```
请评价以下段落的AI味程度（1-10分），并指出3个最"AI"的表达：

[粘贴AI生成的文本]

然后，用更自然的方式改写这3个表达。
```

**关键点**：AI是最好的AI味检测器。让它自己评价自己的输出，往往比你自己判断更准确。
