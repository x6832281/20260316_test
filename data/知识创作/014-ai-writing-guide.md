# AI 写作风格指南：让AI写出"像你写的"文章

**⭐ 星标**：知识创作 | **📅 更新日期**：2026-05-01

---

## 📝 一句话总结

**告别千篇一律的AI味——3步训练AI模仿你的写作风格，从用词习惯、句式节奏到情感表达，全方位打造专属文风。**

---

## 🎯 为什么你的AI写出来的东西"不像你"？

因为AI不知道你是谁。

它不知道你喜欢用短句还是长句，不知道你爱用逗号还是句号，不知道你写悲伤时是直接哭还是只写动作。你每次打开ChatGPT，它都是一张白纸。

要让AI写出"像你写的"文章，你需要做3件事：分析自己的风格→训练AI学习→持续维护。

## 第一步：分析你的写作风格

拿出你写的3-5篇最满意的文章，回答以下问题：

### 1. 句式特征
- 你喜欢用长句还是短句？（平均每句多少字？）
- 你喜欢用排比还是对仗？
- 你喜欢用感叹号还是几乎不用？
- 你的段落有多长？（3-5句一段？还是1-2句一段？）

### 2. 用词偏好
- 你常用哪些词？（列出10个高频词）
- 你从不使用哪些词？（列出5个禁忌词）
- 你喜欢口语化还是书面化？
- 你用成语吗？用得多吗？

### 3. 情感表达
- 你写悲伤时是直接描述感受，还是只写动作？
- 你写快乐时是热烈还是克制？
- 你喜欢在结尾升华主题，还是留白？

### 4. 叙事习惯
- 你喜欢第一人称还是第三人称？
- 你喜欢开门见山还是铺垫再入题？
- 你喜欢用对话还是独白？

## 第二步：训练AI学习你的风格

把第一步的分析结果写成"风格指令"，每次和AI对话时都带上：

```
我的写作风格指南：
1. 句式：用短句，每句不超过15字，偶尔用1-2个长句制造节奏变化
2. 用词：口语化，不用四字成语，不用"宛如""犹如""仿佛"
3. 标点：不用感叹号，用省略号表示停顿
4. 情感：不直接写感受，只写动作和场景
5. 段落：每段3-5句，段与段之间留白
6. 结尾：不升华，不总结，用一个具体的画面收尾
7. 参考风格：类似余华的极简叙事 + 萧红的儿童视角

请严格按照以上风格指南写作。
```

**关键**：这个风格指令要**每次对话都带上**，或者保存为"系统提示词"（System Prompt）。

## 第三步：用Few-shot让AI更精准

除了风格指令，再给AI2-3段你写的范文，让它直接"看到"你的风格：

```
以下是我的写作样例，请学习我的风格：

样例1：
天还没亮，卖早点的已经把摊子支起来了。油锅里的气泡一个接一个地破，白烟直直地往上蹿，被风一扯就散了。

样例2：
他蹲在路边，把烟头按灭在地上。旁边是条河，河水很脏，泛着油光。他看了一会儿河，站起来走了。

样例3：
雨下了三天。院子里的泥没过脚踝，鞋踩下去拔不出来。她光着脚跑出去，脚底板冰得像踩在铁上。

请模仿以上风格，写一段200字的场景描写。
```

Few-shot比纯文字描述有效10倍——因为AI能直接"感受"到你的节奏和用词，而不是靠抽象的规则去猜。

## 💡 风格模板库

建议你维护一个"风格模板库"，包含不同场景的风格指令：

| 场景 | 风格指令关键词 |
|------|--------------|
| 文学评论 | 学术但不枯燥，引用原文，短句为主 |
| 拆书笔记 | 深入浅出，有洞察，有实操练习 |
| 社交媒体 | 口语化，短句，有金句，有互动引导 |
| 小说创作 | 具体场景，动作描写，留白，不写感受 |
| 日常随笔 | 松散，意识流，短段落，有自嘲 |
