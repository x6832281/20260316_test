## 🚀 快速上手（1 分钟搞定）

### 方案一：一键部署到 Vercel（推荐）

1. 访问 [ChatGPT Next Web](https://github.com/ChatGPTNextWeb/ChatGPT-Next-Web)
2. 点击 "Deploy" 按钮
3. 填入你的 OpenAI API Key
4. 等待 1 分钟，部署完成

### 方案二：Docker 本地部署

```bash
docker run -d -p 3000:3000 \
  -e OPENAI_API_KEY=your-api-key \
  -e CODE=your-access-code \
  yidadaa/chatgpt-next-web
```

访问 `http://localhost:3000` 即可使用。

### 方案三：安装为 PWA

部署完成后：
1. 用浏览器访问你的部署地址
2. 点击浏览器地址栏的"安装"按钮
3. 安装后像原生 App 一样使用

## 📚 核心功能详解

### 1. 面具功能（Mask）—— 快速切换写作场景

**使用场景**：不同写作任务需要不同的 AI 人设。

**内置面具**：
- **翻译官**：专业中英互译
- **润色专家**：优化文风表达
- **写作助手**：帮你构思创作
- **代码助手**：帮你写代码

**自定义面具**：
1. 点击"新建面具"
2. 设置名称、描述、系统提示词
3. 保存后一键切换

**实用技巧**：
- 为不同项目创建不同面具（如：小说写作、论文写作）
- 在面具中预设好提示词，省去每次输入

### 2. 提示词模板

**使用场景**：快速调用常用提示词。

**内置模板**：
- 写作类：文章大纲、段落扩展、风格转换
- 翻译类：中英互译、学术翻译
- 润色类：语法修正、表达优化

**自定义模板**：
1. 点击"提示词管理"
2. 添加你的常用提示词
3. 对话时一键调用

### 3. 对话管理

**使用场景**：管理多个写作项目。

**功能**：
- 按主题创建多个对话
- 对话历史本地存储
- 支持导出为 Markdown

**实用技巧**：
- 为每个写作项目创建独立对话
- 重要对话可以"置顶"方便查找
- 定期导出备份重要对话

### 4. 模型切换

**支持的模型**：
- OpenAI GPT-4、GPT-3.5
- Claude
- Google Gemini
- 百度文心一言
- 阿里通义千问

**操作**：对话时点击模型名称即可切换。

## 💡 进阶使用技巧

### 技巧一：配置访问密码

如果部署在公网，建议设置访问密码：

1. 在 Vercel 环境变量中添加 `CODE=your-password`
2. 访问时需要输入密码才能使用

### 技巧二：使用 Markdown 语法

ChatGPT Next Web 完美支持 Markdown：

- 用 `#` 创建标题
- 用 `**` 加粗重点
- 用 `` ` `` 标记代码
- 用 `>` 创建引用

**实用技巧**：
- 让 AI 用 Markdown 格式输出，方便后续编辑
- 代码块会自动高亮显示

### 技巧三：快捷键操作

- `Ctrl + N`：新建对话
- `Ctrl + Shift + N`：新建面具对话
- `Ctrl + Enter`：发送消息
- `Ctrl + ,`：打开设置

## ⚠️ 使用注意事项

### 避坑指南

1. **API Key 安全**
   - 不要把 API Key 明文写在代码里
   - 使用环境变量配置

2. **成本控制**
   - GPT-4 调用费用较高
   - 建议设置每日费用上限

3. **网络问题**
   - 国内访问 Vercel 可能需要代理
   - 建议使用自定义域名

4. **数据备份**
   - 数据存储在浏览器本地
   - 清除浏览器数据会丢失对话历史
   - 建议定期导出备份

## 🔧 常见问题解答

**Q：和 LobeChat 有什么区别？**
A：ChatGPT Next Web 更轻量、部署更简单，但功能相对单一。LobeChat 功能更丰富（插件市场、知识库），但部署稍复杂。

**Q：需要付费吗？**

**Q：支持哪些平台？**
A：Web 端（所有浏览器）、PWA（手机/电脑/平板）。

**Q：数据安全吗？**
A：数据存储在浏览器本地，不经过第三方服务器，非常安全。
