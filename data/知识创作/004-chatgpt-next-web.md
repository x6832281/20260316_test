## 📝 一句话总结

GitHub 上最火的 ChatGPT 私有化部署方案，一键拥有跨平台的 AI 写作助手，支持手机/电脑/平板。

## 📋 详细介绍

ChatGPT Next Web（简称 NextChat）是 GitHub 上 Star 数最高的 ChatGPT 网页客户端之一，以极简部署和跨平台体验著称：

- 一键部署：Vercel 上 1 分钟完成部署，无需服务器
- 全平台 PWA：支持安装到手机/电脑/平板，像原生 App 一样使用
- 隐私安全：所有数据存储在本地浏览器，不经过第三方服务器
- 多模型兼容：支持 OpenAI、Azure、Google AI、百度文心等
- Markdown 渲染：完美支持代码高亮、LaTeX 公式、表格等
- 提示词模板：内置丰富的写作提示词，一键调用
- 面具功能（Mask）：预设角色和提示词，快速切换写作场景

对于想要一个轻量、私密、随时可用的 AI 写作助手的人来说，这是最佳选择。

## 🎯 个人评价

优点：部署极简、跨平台、隐私保护好、社区庞大。缺点：功能相对单一，不支持插件。适合人群：个人写作者、隐私敏感用户。

## 🔧 技术栈

- React / Next.js
- TypeScript
- PWA
- Vercel 部署
