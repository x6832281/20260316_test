## 🚀 快速上手（5 分钟搞定）

### 方案一：一键部署到 Vercel（推荐）

1. 访问 [LobeChat Vercel 部署页面](https://github.com/lobehub/lobe-chat#-deploy-with-vercel)
2. 点击 "Deploy" 按钮
3. 填入你的 OpenAI API Key
4. 等待部署完成，即可使用

### 方案二：Docker 本地部署

```bash
# 拉取镜像
docker pull lobehub/lobe-chat

# 启动容器
docker run -d -p 3210:3210 \
  -e OPENAI_API_KEY=your-api-key \
  --name lobe-chat \
  lobehub/lobe-chat
```

访问 `http://localhost:3210` 即可使用。

### 方案三：本地开发

```bash
# 克隆项目
git clone https://github.com/lobehub/lobe-chat.git
cd lobe-chat

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

## 📚 核心功能详解

### 1. 多模型切换（最实用）

**使用场景**：不同写作任务用不同模型。

**支持的模型**：
- **GPT-4**：适合深度写作、复杂逻辑
- **Claude**：适合长文创作、代码理解
- **DeepSeek**：性价比高，中文优秀
- **Gemini**：多模态能力强
- **Qwen**：阿里出品，中文理解好

**操作步骤**：
1. 在设置中配置多个 API Key
2. 对话时点击模型名称切换
3. 不同对话可以用不同模型

**实用技巧**：
- 大纲用 DeepSeek（便宜），成稿用 GPT-4（质量高）
- 翻译用 Claude（表达自然），润色用 GPT-4（精准）

### 2. 知识库功能

**使用场景**：让 AI 基于你的素材写作。

**操作步骤**：
1. 点击侧边栏"知识库"
2. 上传你的文档（支持 PDF、Word、TXT）
3. 等待系统自动向量化
4. 对话时选择关联的知识库

**实用技巧**：
- 把你的写作素材、参考资料都上传到知识库
- 对话时说"基于知识库中的素材，帮我写..."
- 定期更新知识库，保持素材新鲜

### 3. 插件市场

**使用场景**：扩展 AI 的能力。

**推荐插件**：
- **搜索插件**：让 AI 实时搜索最新资料
- **阅读插件**：让 AI 阅读网页/文档
- **画图插件**：让 AI 生成配图
- **代码插件**：让 AI 执行代码

**操作步骤**：
1. 点击侧边栏"插件市场"
2. 浏览并安装需要的插件
3. 对话时启用插件

### 4. 角色市场

**使用场景**：一键切换 AI 的"人设"。

**推荐角色**：
- **写作助手**：帮你构思、写作、润色
- **翻译官**：高质量中英互译
- **润色专家**：优化文风、提升表达
- **创意写作**：写小说、剧本、诗歌

**操作步骤**：
1. 点击侧边栏"角色市场"
2. 选择需要的角色
3. 对话时 AI 会以该角色身份回答

## 💡 进阶使用技巧

### 技巧一：项目化管理写作会话

LobeChat 支持按项目/主题管理会话：

1. 创建"项目"（如：小说创作、论文写作）
2. 在项目下创建多个会话（如：第一章、第二章）
3. 每个会话独立管理，互不干扰

### 技巧二：使用 TTS 朗读检查语感

写完文章后，用 TTS 功能朗读：

1. 选中生成的文本
2. 点击"朗读"按钮
3. 听一遍，检查语感是否流畅

**实用技巧**：
- 朗读时注意听"不自然"的地方，那就是需要修改的地方
- 可以调整语速，慢速朗读更容易发现问题

### 技巧三：导出对话记录

把有价值的对话导出保存：

1. 点击对话右上角"导出"
2. 选择格式（Markdown、JSON）
3. 保存到本地，方便以后查阅

## ⚠️ 使用注意事项

### 避坑指南

1. **API Key 安全**
   - 不要把 API Key 分享给别人
   - 建议使用环境变量，不要硬编码

2. **成本控制**
   - 多模型切换容易忘记切换回便宜模型
   - 建议设置每日费用上限

3. **数据隐私**
   - 本地部署的数据完全在你手里
   - Vercel 部署的数据会经过第三方服务器

4. **网络问题**
   - 国内访问可能需要代理
   - 建议使用国内模型（DeepSeek、Qwen）避免网络问题

## 🔧 常见问题解答

**Q：和直接用 ChatGPT 有什么区别？**
A：LobeChat 支持多模型切换、知识库、插件市场、角色市场，功能远比 ChatGPT 丰富。而且是开源的，可以自己部署。

**Q：需要付费吗？**
A：LobeChat 本身免费开源，但需要自己准备大模型的 API Key（会产生调用费用）。

**Q：支持哪些平台？**
A：Web 端（所有浏览器）、桌面端（Windows/Mac/Linux）、移动端（PWA）。

**Q：数据安全吗？**
A：本地部署的数据完全在你手里，非常安全。Vercel 部署的数据会经过第三方服务器，敏感数据建议本地部署。
