# LobeChat —— 现代化 AI 写作对话平台

**⭐ 星标**：48.0k | **📅 更新日期**：2026-04-27

## 📝 一句话总结

颜值最高的开源ChatGPT网页客户端，支持多模型、插件市场、知识库，是AI写作的绝佳前端。

## 📋 详细介绍

LobeChat 是一个开源的现代化 AI 对话平台，以其精美的 UI 设计和丰富的功能生态闻名。对知识创作者来说，它提供了远超普通 ChatGPT 界面的写作体验：

- 多模型支持：可同时使用 GPT-4、Claude、Gemini、DeepSeek、Qwen 等
- 插件市场：内置搜索、阅读、画图等插件，辅助写作资料搜集
- 知识库功能：上传文档建立专属知识库，让 AI 基于你的素材写作
- 角色市场：一键切换写作助手、润色专家、翻译官等角色
- 会话组织：按项目/主题管理写作会话
- TTS 语音：朗读生成的文章，检查语感流畅度

项目支持一键部署（Vercel/Docker），5分钟即可拥有专属的 AI 写作工作站。

## 🎯 个人评价

优点：颜值高、功能全、部署简单、社区活跃。缺点：部分高级功能需要自己配置 API。适合人群：所有需要AI辅助写作的用户。

## 🔧 技术栈

- React / Next.js
- TypeScript
- Ant Design
- PWA 离线支持
