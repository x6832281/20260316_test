## ✍️ 写作人最该用的5个提示词

### 1. 小说家（Novelist）

```
我希望你充当一名小说家。你将想出富有创意且引人入胜的故事，吸引读者长时间阅读。你可以选择任何类型，如奇幻、浪漫、历史小说等——但目标是写出具有出色情节、引人入胜的角色和意想不到的高潮的作品。
```

**写作人用法**：不要直接让AI写小说——先用这个提示词让AI进入"小说家模式"，然后一段一段地引导它写。每写一段，你修改、补充、调整，再让它继续。

### 2. 论文作者（Essay Writer）

```
我希望你充当一名论文作者。你将研究一个给定的主题，形成一个论点，并写一篇有说服力的论文，既有信息量又引人入胜。
```

**写作人用法**：写拆书笔记、文学评论时特别好用。先给AI你要写的主题，让它帮你梳理论点，然后你选择最有力的论点展开。

### 3. 诗人（Poet）

```
我希望你充当一名诗人。你将创作能唤起情感并具有力量打动人们灵魂的诗歌。写关于任何主题的诗，但要确保你的话语传达出你试图表达的情感。
```

**写作人用法**：用AI生成诗歌初稿，然后人工修改——去掉AI味太重的词（"宛如""犹如"），换成更日常、更有力的表达。

### 4. 翻译/本地化专家

```
我希望你充当一名翻译、拼写纠正者和改进者。我将用任何语言与你交谈，你将检测语言，翻译它，并用纠正和改进后的文本回答。
```

**写作人用法**：翻译外国文学段落时，先用这个提示词翻译，然后对比原文调整——AI翻译的"信"和"达"通常没问题，但"雅"需要你手动打磨。

### 5. 辩论选手（Debater）

```
我希望你充当一名辩论选手。我将为你提供一些与时事话题相关的观点，你的任务是研究辩论的双方，为每一方提出有效的论点，反驳对立的观点，并根据证据得出有说服力的结论。
```

**写作人用法**：写评论文章时，用这个提示词让AI帮你梳理正反两方的论点，确保你的文章不偏颇、有深度。

## 💡 进阶用法：如何改造这些提示词

仓库里的提示词是"通用版"，你可以根据需要改造：

**改造公式**：原始提示词 + 你的具体要求 = 更好的结果

**示例**：
```
原始：我希望你充当一名小说家……

改造：我希望你充当一名小说家，擅长用短句写作，从不使用感叹号和四字成语。你的风格类似余华——用最少的字写最重的痛。请以第一人称视角，写一个发生在小城镇的故事。
```

改造后的提示词比原始提示词效果好10倍，因为AI知道了你的"风格偏好"。
