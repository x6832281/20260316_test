## 🔍 什么是 AI 写作 SKILL？

SKILL 不是单一的提示词，而是一套组合拳——它包含定制化的系统指令、工作流程和知识库，让 AI 在特定写作场景下表现得更专业。

## 📖 四大 SKILL 详解

### SKILL 1：长篇连贯性写作 Skill

| 维度 | 详情 |
|------|------|
| ⭐ 社区热度 | 极高 |
| 🎯 适用场景 | 万字以上长文、连载小说、系列文章 |

**一句话说明**：让 AI 写长篇内容时保持逻辑连贯、人物一致、设定不崩。

**核心原理**：长篇写作最大的挑战是"AI 会忘记"。这个 SKILL 通过以下机制解决：
1. **章节摘要链**：每写完一章，让 AI 自动生成摘要，下一章写作时先加载摘要
2. **人物状态追踪**：维护一个动态的人物状态表，每次写作前更新
3. **设定锁定机制**：关键设定一旦确定就写入"不可变清单"

**实战提示词模板**：

在每次开始新章节时，使用如下系统指令：
```
你现在是一位小说续写助手。在开始写作之前，请先回顾以下内容：

【前情摘要】
[粘贴上一章摘要]

【人物当前状态】
- 张三：刚从警察局出来，情绪低沉，手腕受伤
- 李四：在医院照顾受伤的张三，内心愧疚

【不可变设定】
- 故事时间：2045 年
- 核心规则：所有 AI 必须遵守"三定律"

请根据以上信息，续写下一章节。
```

**效果验证**：使用此 SKILL 后，10 万字长文中 AI 忘记设定的概率从 ~40% 降低到 ~5%。

### SKILL 2：人设卡 Skill

| 维度 | 详情 |
|------|------|
| ⭐ 社区热度 | 高 |
| 🎯 适用场景 | 小说角色塑造、人物对话、角色视角写作 |

**一句话说明**：给 AI 一张详细的人物卡片，让角色告别"千篇一律"。

人设卡通过以下维度让角色"活"起来：
1. **基础信息**：姓名、年龄、职业、外貌
2. **性格光谱**：多维度描述（不是简单的"内向/外向"）
3. **行为模式**：遇到特定情况时的典型反应
4. **语言风格**：口头禅、惯用句式、说话节奏
5. **隐藏设定**：内心恐惧、未实现的渴望、秘密

**实战示例**：
```
【角色卡：陈默】
- 基础：32 岁，前刑警，现私家侦探，左脸有刀疤
- 性格：对外冷漠但心软，嘴硬，从不解释自己的行为
- 行为模式：紧张时摸刀疤 / 只喝浓茶 / 见小孩会微笑
- 语言风格：说话不超过 15 字 / 惯用语："知道了""不关你事"
- 隐藏设定：女儿三年前失踪，这是他做侦探的真正原因

请严格按照以上人设来塑造角色的一言一行。
```

### SKILL 3：大纲结构化 Skill

| 维度 | 详情 |
|------|------|
| ⭐ 社区热度 | 高 |
| 🎯 适用场景 | 长文规划、论文写作、课程设计 |

**一句话说明**：用结构化大纲锁定文章骨架，让 AI 写出的内容逻辑严密、层次分明。

核心理念："先骨架，后血肉"——
- **三幕结构**：开端 → 发展/冲突 → 结局/总结
- **金字塔原理**：核心观点 → 分论点 → 支撑论据

实战三步走：
1. 生成结构化大纲 → 2. 逐段填充 → 3. 检查衔接

### SKILL 4：提示词工程 Skill

| 维度 | 详情 |
|------|------|
| ⭐ 社区热度 | 极高 |
| 🎯 适用场景 | 所有 AI 写作场景 |
| 🔗 参考来源 | https://github.com/f/awesome-chatgpt-prompts |

**一句话说明**：提示词是 AI 时代的"编程语言"，写好提示词 = 写好文章。

**黄金公式**：角色 + 任务 + 约束 + 示例 = 高质量输出

**从新手到高手的进化**：
- 新手版："帮我写一个爱情故事。" → 千篇一律
- 进阶版：指定角色（小说家）+ 任务（写开头）+ 约束（300字/第一人称/开头悬念）+ 示例（村上春树风格） → 有细节、有风格
