## 📌 一句话总结

**去AI味的终极目标不是"去掉AI味"，而是"注入人味"——让文字有温度、有态度、有个性。**

---

## 🎯 什么是"人味"？

人味不是"口语化"，不是"接地气"，而是一种**独特的个人风格**。

每个人说话都有自己的特点：
- 有人喜欢用反问句
- 有人喜欢用比喻
- 有人喜欢用短句
- 有人喜欢用长句
- 有人喜欢用幽默
- 有人喜欢用严肃

AI写的文字之所以"没有味道"，是因为它用的是"平均风格"——把所有人的风格平均一下，得到的就是"没有风格"。

去AI味的终极目标，是给AI注入**你的风格**。

---

## 🎨 四种常见的人味风格

### 风格一：朋友聊天型

**特点**：
- 多用"你""我"
- 多用短句
- 多用口语化表达
- 多用反问句

**Prompt模板**：
```
请用朋友聊天的风格改写以下内容：
- 多用"你""我"，少用"我们""大家"
- 多用短句，每句话不超过20字
- 多用口语化表达（"说白了""其实""你猜怎么着"）
- 多用反问句（"你有没有发现？""这不是很明显吗？"）

原文：
[粘贴内容]
```

**示例对比**：
```
AI味：在当今数字化时代，人工智能技术正在深刻改变着各行各业的运作模式。

人味：你有没有发现，最近身边用AI的人越来越多了？写文案的、做设计的、甚至写代码的——好像一夜之间，AI就成了标配。
```

---

### 风格二：老司机经验型

**特点**：
- 多用"我试过""我发现"
- 多用具体数据和案例
- 多用"踩坑""避坑"等经验表达
- 多用"说实话""老实说"

**Prompt模板**：
```
请用老司机分享经验的风格改写以下内容：
- 多用"我试过""我发现""我踩过坑"
- 多用具体数据（"我试了3个工具，就这个能用"）
- 多用"踩坑""避坑""血泪教训"等经验表达
- 多用"说实话""老实说""不骗你"

原文：
[粘贴内容]
```

**示例对比**：
```
AI味：该工具具有强大的功能特性，能够有效提升工作效率。

人味：说实话，这工具我用了三个月，最让我惊喜的是它的批量处理功能——以前要手动改一天的文档，它半小时就搞定了。
```

---

### 风格三：犀利吐槽型

**特点**：
- 多用反讽和吐槽
- 多用"说白了""翻译一下"
- 多用对比和反差
- 敢于下结论

**Prompt模板**：
```
请用犀利吐槽的风格改写以下内容：
- 多用反讽和吐槽（"这不就是...吗？""说白了就是..."）
- 多用"翻译一下""说人话"等表达
- 多用对比和反差（"听起来高大上，其实就是..."）
- 敢于下结论，不要"辩证看待"

原文：
[粘贴内容]
```

**示例对比**：
```
AI味：该方案具有多方面的优势，同时也存在一些需要改进的地方。

人味：说白了，这方案就是把简单的事情搞复杂了。听起来高大上，其实就是换了个名字的老套路。
```

---

### 风格四：故事叙述型

**特点**：
- 用具体场景开头
- 用对话推动情节
- 用细节展示而非直接陈述
- 留白给读者思考

**Prompt模板**：
```
请用讲故事的风格改写以下内容：
- 用具体场景开头（"上周三下午，我正在..."）
- 用对话推动情节（"他跟我说了一句话，我愣了半天"）
- 用细节展示（不要说"他很紧张"，而是写"他的手在抖"）
- 结尾留白，不要点题

原文：
[粘贴内容]
```

**示例对比**：
```
AI味：学习AI写作需要掌握正确的方法，才能取得良好的效果。

人味：上周有个读者私信我，说他用AI写了一篇文章，发出去被骂惨了。我问他怎么写的，他说"我就让AI帮我写一篇关于AI的文章"。我听完就明白了——他犯了一个很多人都会犯的错。
```

## 🛠️ 人味注入的五个技巧

### 技巧一：注入个人经历

**方法**：在文章中加入你的真实经历。

**Prompt**：
```
请在以下内容中加入1-2个个人经历：
- 用"我上周试了一个方法""我一个朋友告诉我"等表达
- 经历要具体（有时间、地点、人物、结果）
- 经历要真实，不要编造

原文：
[粘贴内容]
```

### 技巧二：注入个人观点

**方法**：在文章中加入你的明确观点。

**Prompt**：
```
请在以下内容中加入个人观点：
- 用"我认为""我的经验是""说白了"等表达
- 观点要明确，不要"辩证看待"
- 观点要有依据（数据、经历、逻辑）

原文：
[粘贴内容]
```

### 技巧三：注入情感温度

**方法**：在文章中加入真实的情感表达。

**Prompt**：
```
请在以下内容中加入情感温度：
- 用"说实话，我当时挺失望的""我也没想到效果这么好"等表达
- 情感要真实，不要"我们深感荣幸"等官方腔调
- 允许表达不确定（"我不确定""我还在摸索"）

原文：
[粘贴内容]
```

### 技巧四：注入独特比喻

**方法**：在文章中加入你自己的比喻。

**Prompt**：
```
请在以下内容中加入2-3个独特比喻：
- 比喻要具体（不要"像阳光一样温暖"，而是"像冬天里喝到的第一口热奶茶"）
- 比喻要原创，不要用经典比喻
- 比喻要贴合上下文

原文：
[粘贴内容]
```

### 技巧五：注入节奏变化

**方法**：让文章的节奏有快有慢。

**Prompt**：
```
请调整以下内容的节奏：
- 重要观点用短句（"这就是问题所在。"）
- 解释说明用长句
- 关键转折用单独成段
- 允许用省略号、破折号等标点制造停顿

原文：
[粘贴内容]
```

## 📝 人味注入自检表

改完文章后，用这个表检查：

- [ ] 有没有加入个人经历？
- [ ] 有没有加入明确观点？
- [ ] 有没有加入情感温度？
- [ ] 有没有加入独特比喻？
- [ ] 有没有节奏变化（长短句交替）？

如果以上5项中有3项以上"有"，说明文章有人味了。
