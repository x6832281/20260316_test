### 博主简介

漫士沉思录专注用 AI 辅助构建宏大世界观——从奇幻世界的地理设定到科幻世界的科技树。

| 平台 | 链接 |

| 知乎 | https://www.zhihu.com/people/ （搜索"漫士沉思录"） |

### 核心方法论：世界观九宫格

| 物理法则 | 社会结构 | 历史事件 |
|---------|---------|---------|
| 世界基本物理规则？ | 权力如何分配？ | 改变世界的关键事件？ |
| 技术/魔法水平 | 文化习俗 | 冲突来源 |
| 资源分布？ | 人们相信什么？ | 主要矛盾是什么？ |
| 生态/物种 | 经济体系 | 当前危机 |

**用 AI 填充九宫格提示词**：
```
请帮我为以下世界观填充"九宫格"：
【简要描述】一个蒸汽朋克风格的世界，魔法和蒸汽机械并存。
请按顺序回答：物理法则、技术魔法水平、资源分布、生态物种、
社会结构、文化习俗、经济体系、历史事件、当前危机。
```

**漫士的创作准则**：
1. **设定服务于故事**：不要为了炫世界观而设定，每个设定都要推动情节
2. **用 AI 检查一致性**：设定完成后，让 AI 检查是否有内部矛盾
3. **留白给读者想象**：好世界观是"冰山"，露出 20%，隐藏 80%

## 📖 陈行之（皮特）：故事内核

### 博主简介

陈行之（皮特）专注研究"故事的内核"——为什么有些故事让人欲罢不能，有些让人无感。

| 平台 | 链接 |
|------|------|
| B站 | https://space.bilibili.com/ （搜索"陈行之"） |
| 知乎 | https://www.zhihu.com/people/ （搜索"陈行之"） |

### 陈行之的故事内核四要素

**1. 欲望引擎**

每个好故事都有一个"欲望引擎"在驱动：
- 主角想要什么？（明确的目标）
- 为什么现在必须得到？（紧迫性）
- 得不到会怎样？（代价）

**用 AI 检查欲望引擎**：
```
请分析以下故事设定的"欲望引擎"是否足够强：
[粘贴你的故事设定]
如果不够强，请给出 3 个强化方案。
```

**2. 冲突升级**

陈行之认为，好故事不是"有冲突"，而是"冲突在升级"：
- 第一幕冲突：外部障碍
- 第二幕冲突：内部矛盾
- 第三幕冲突：终极抉择

**3. 情感锚点**

读者不是被情节打动的，是被情感打动的。每个关键情节都要有一个"情感锚点"——让读者产生共鸣的情感瞬间。

**4. 主题落地**

好故事都有一个"主题"——不是贴在表面的口号，而是通过故事自然浮现的"道理"。

### 张然冉的"图助文"方法论

**应用一：角色可视化**
用 AI 绘画工具（Midjourney/Stable Diffusion）生成角色形象，帮助自己和读者建立具象认知。

**应用二：场景氛围图**
为关键场景生成氛围图（不一定是精确插图，而是色调、构图、氛围的参考）。

**应用三：世界观概念图**
为独特的设定（特殊建筑、道具、生物）生成概念图。

**张然冉的推荐工具**：
- **Midjourney**：画质最好，适合角色和场景
- **Stable Diffusion**：开源免费，可本地运行
- **DALL-E 3**：理解力强，适合复杂描述

## 📖 立党老师：素材搜集

### 博主简介

立党老师专注用 AI 高效搜集整理写作素材，解决"肚里没货写不出"的困境。

| 平台 | 链接 |
|------|------|
| B站 | https://space.bilibili.com/ （搜索"立党老师"） |
| 知乎 | https://www.zhihu.com/people/ （搜索"立党老师"） |

### 立党老师的素材搜集三法

**方法一：AI 定向搜索整理**

提示词模板：
```
请帮我搜集关于[主题]的素材，要求：
1. 列出 5 个相关的真实案例或数据
2. 每个案例标明来源（如果能查到的话）
3. 按"能用在文章开头抓人"的标准筛选
4. 优先选择近期（一年内）的案例
```

**方法二：AI 跨领域关联**

把看似无关的领域素材关联起来：
```
请将[领域A]和[领域B]进行关联，找出 5 个可以互相印证的案例或观点。
```

**方法三：AI 素材卡片系统**

立党老师建议为每个素材建立卡片：
- **事实**：这个素材的核心事实是什么？
- **观点**：作者想用这个事实说明什么？
- **适用场景**：这个素材适合用在什么类型的文章中？
- **来源**：原始出处，方便引用
