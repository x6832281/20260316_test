## 📌 一句话总结

**去AI味的核心在Prompt——你给AI什么指令，它就给你什么味道的文字。**

## 🎯 Prompt调教的基本原则

### 原则一：明确"写给谁看"

AI默认写给"所有人"看，所以它会用最保险、最中庸的表达。你必须告诉它具体读者是谁。

**错误Prompt**：
```
帮我写一篇关于AI写作的文章
```

**正确Prompt**：
```
帮我写一篇关于AI写作的文章，读者是刚开始接触AI写作的新手，他们最担心的是"AI写出来的东西不像人写的"。用朋友聊天的语气，不要用学术腔。
```

### 原则二：明确"什么风格"

AI默认用"标准中文"写作，但"标准中文"往往就是"AI味中文"。你必须指定具体的风格。

**错误Prompt**：
```
帮我改写这段话
```

**正确Prompt**：
```
请用口语化、接地气的风格改写这段话，像朋友聊天一样自然。去掉所有书面语和官方腔调。
```

### 原则三：明确"不要什么"

告诉AI"不要做什么"比"要做什么"更重要。

**错误Prompt**：
```
让这段话更自然
```

**正确Prompt**：
```
改写这段话，要求：
1. 不要用"在当今""随着""综上"等万能开头
2. 不要用"首先/其次/最后"的结构
3. 不要用超过2个形容词修饰同一个名词
4. 不要用"以XX为例"的万能例子
```

## 🎯 不同场景的Prompt技巧

### 场景一：写公众号文章

```
帮我写一篇公众号文章，要求：
1. 标题要有悬念或反差（如"AI写作最大的坑，不是写不好"）
2. 开头用一个具体场景或问题切入
3. 每段不超过3句话
4. 多用"你""我"，少用"我们""大家"
5. 结尾留一个互动问题（如"你遇到过这种情况吗？"）
6. 不要用"在当今""随着""综上"等万能表达
```

### 场景二：写产品介绍

```
帮我写一篇产品介绍，要求：
1. 不要用"强大的""高效的""智能的"等堆砌形容词
2. 用具体数据替代形容词（如"处理速度提升3倍"而非"处理速度极快"）
3. 用用户真实评价替代官方宣传语
4. 用"我试了XX，发现..."的个人体验视角
5. 不要用"我们致力于""我们深感荣幸"等官方腔调
```

### 场景三：写技术教程

```
帮我写一篇技术教程，要求：
1. 不要用"首先/其次/最后"的八股结构
2. 用"先做XX，然后XX，最后XX"的自然流程
3. 每步操作都要说明"为什么要这样做"
4. 用"我踩过的坑"替代"注意事项"
5. 用"你可能会遇到的问题"替代"常见问题"
```

### 场景四：写故事/小说

```
帮我写一个故事，要求：
1. 不要用"在那个风和日丽的下午"等万能开头
2. 用对话推动情节，不要用大段心理描写
3. 用动作和细节展示人物性格，不要直接说"他是一个XX的人"
4. 允许不完美——人物可以说错话、做错事
5. 结尾不要"点题"，留白给读者思考
```

## ⚠️ Prompt调教的常见误区

### 误区一：加"请写得自然一点"

这太模糊了，AI不知道什么是"自然"。你必须具体说明什么是"自然"。

**正确做法**：用具体的规则替代模糊的要求。

### 误区二：加"请用中文写作"

AI本来就用中文写作，加这个没用。你需要的是"用什么样的中文"。

**正确做法**：指定具体的风格（口语化、书面语、文艺腔等）。

### 误区三：一次性给太多要求

如果你一次性给AI 20条要求，它可能会顾此失彼。

**正确做法**：每次只给3-5条核心要求，分多次修改。

### 误区四：只改一次就满意

去AI味往往需要多轮修改。第一轮去掉明显的AI味，第二轮注入人味，第三轮精修细节。

**正确做法**：至少改3轮，每轮聚焦不同的问题。

- [ ] 是否给了具体的规则而非模糊的要求？
- [ ] 是否控制在3-5条核心要求内？
