## 📖 精选项目详解

### 1. inkOS —— AI 写作操作系统

| 维度 | 详情 |
|------|------|
| ⭐ 星标 | 12.5k+ |
| 🏷️ 语言 | TypeScript / Python |
| 📅 最近更新 | 2026-04 |
| 🔗 仓库 | https://github.com/inkos/inkos |

**项目简介**：

inkOS 是一个专为写作者设计的 AI 工作台，将"写作"这件事拆解为模块化的流程——从灵感搜集、大纲规划、章节写作到润色发布，全流程覆盖。

**核心功能**：
- **多模型支持**：可接入 GPT-4、Claude 3.5、Gemini 2.0 等多个 AI 模型
- **项目式写作管理**：按书/文章组织写作项目，支持章节管理和版本对比
- **智能大纲生成**：输入一句话描述，自动生成多层级大纲
- **写作风格锁定**：上传你的作品，AI 学习你的风格后辅助写作
- **一键导出**：支持 Markdown、PDF、EPUB 等多格式导出

**为什么推荐给萌新**：
inkOS 把 AI 写作的门槛降到了最低——不需要学提示词工程，一个平台搞定全部写作流程。

### 2. AI-Writer —— 开源 AI 长文写作助手

| 维度 | 详情 |
|------|------|
| ⭐ 星标 | 8.3k+ |
| 🏷️ 语言 | Python |
| 📅 最近更新 | 2026-03 |
| 🔗 仓库 | https://github.com/ai-writer/ai-writer |

**项目简介**：AI-Writer 专注于解决"用 AI 写长文容易跑偏"的难题。通过章节锁定、上下文管理和情节一致性检查，让 AI 辅助创作上万字的长文成为可能。

**核心功能**：
- **上下文窗口管理**：自动管理超长对话，确保 AI 不会"忘记"前面的内容
- **情节一致性检查**：自动检测人物性格、时间线、设定是否前后矛盾
- **本地运行**：支持 Ollama 本地模型，无需联网

### 4. LangChain —— 可编程的写作流水线

| 维度 | 详情 |
|------|------|
| ⭐ 星标 | 85k+ |
| 🔗 仓库 | https://github.com/langchain-ai/langchain |

LangChain 的 Chain 机制特别适合构建"写作流水线"——搜集资料 → 分析 → 列大纲 → 写作 → 润色 → 检查，每个步骤用链式调用串联。
