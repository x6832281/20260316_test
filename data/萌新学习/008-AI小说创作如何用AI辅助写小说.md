## 📌 一句话总结

**AI 是你的写作搭档，不是你的替写机器——用它来构思、扩展、润色，你来把控故事灵魂。**

---

## 🔍 为什么用 AI 写小说？

写小说最难的不是写，而是"想"——想人物、想情节、想细节。AI 最擅长的就是帮你"想"，给你灵感，帮你突破卡文的困境。

### AI 在小说创作中的角色

| 阶段 | AI 能做什么 | 你需要做什么 |
|------|------|------|
| **构思** | 生成故事概念、人物设定 | 选择和筛选方向 |
| **大纲** | 列出章节大纲、情节线 | 调整逻辑和节奏 |
| **扩展** | 把简短大纲扩展成段落 | 把控风格和语气 |
| **润色** | 修改病句、优化表达 | 保持个人风格 |
| **查漏** | 检查情节漏洞、前后矛盾 | 最终审核把关 |

## 📖 AI 小说创作全流程

### 第一步：构思故事概念

**提示词示例**：
```
我想写一部都市悬疑小说，请帮我生成3个不同的故事概念，
每个概念包含：一句话概括、主角设定、核心冲突、独特卖点。
```

**AI 可能的回答**：
1. 《消失的邻居》：一个程序员发现楼上邻居凭空消失，但所有人都不记得这个人存在过。主角：28岁程序员，社交恐惧。核心冲突：记忆与真相。独特卖点：每个人对"消失者"的记忆都不同。
2. 《深夜外卖》：一个外卖骑手发现，每天凌晨3点的同一单外卖，送到的地址每次都不一样，但收件人都是同一个名字。主角：25岁骑手，夜班打工人。核心冲突：现实与超自然。独特卖点：都市传说与外卖文化的碰撞。
3. 《第七层》：一栋6层老楼突然出现了第7层，只有特定的人能看到。主角：30岁物业管理员。核心冲突：正常与异常。独特卖点：空间悬疑+社会隐喻。

### 第二步：塑造人物

**提示词示例**：
```
请帮我为《消失的邻居》的主角创建详细的人物卡，包含：
姓名、年龄、职业、性格特点（3个优点3个缺点）、
成长背景、口头禅、一个标志性习惯、内心最大的恐惧。
```

### 第三步：规划情节线

**提示词示例**：
```
请帮我为《消失的邻居》规划10章的大纲，
要求：每章末尾有一个悬念或转折，
整体节奏是"慢热→加速→高潮→反转"，
第5章有一个重大揭示，第9章有最大反转。
```

### 第四步：逐章写作

**提示词示例**：
```
请根据以下大纲，帮我写第1章的开头500字，
要求：用第一人称，语气偏冷幽默，以一个日常场景开头，
但暗示有些不对劲。

第1章大纲：主角在加班后回家，发现楼上邻居的门口有一双鞋，
但第二天鞋消失了，邻居似乎从未存在过。
```

### 第五步：润色和统一风格

**提示词示例**：
```
请帮我统一以下两段文字的叙事风格，
让它们读起来像同一个人写的，保持冷幽默的语气：
[段落1]
[段落2]
```

## ⚠️ 注意事项

### 注意 1：AI 可能"跑偏"
**问题**：AI 生成的情节可能不符合你的设想
**解决**：每次只让 AI 生成一小段，你确认后再继续

### 注意 2：保持个人风格
**问题**：AI 写的文字有"AI味"，缺乏个人特色
**解决**：AI 生成初稿，你必须用自己的风格改写

### 注意 3：注意原创性
**问题**：AI 可能生成与其他作品相似的情节
**解决**：核心创意必须来自你自己，AI 只负责扩展
