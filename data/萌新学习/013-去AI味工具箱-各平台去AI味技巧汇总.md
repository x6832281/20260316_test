---

## 🎯 为什么要分平台？

去AI味不是"一招鲜吃遍天"。不同平台的读者不同、风格不同、规则不同，去AI味的方法也不同。

- **公众号**：读者期待深度内容，允许长文，但要有人味
- **小红书**：读者期待干货和颜值，要求短平快，要接地气
- **知乎**：读者期待专业分析，允许长文，但要有深度
- **抖音/短视频**：读者期待快节奏，要求抓眼球，要刺激
- **B站**：读者期待有趣和有用，允许长内容，但要有梗

你必须根据平台特点，调整去AI味的策略。

## 📱 公众号去AI味技巧

### 公众号的特点
- 读者愿意读长文（1500-3000字）
- 读者期待深度和干货
- 读者喜欢有温度的内容
- 读者讨厌营销腔和官方腔

### 公众号去AI味Prompt
```
帮我写一篇公众号文章，要求：

1. 标题：要有悬念或反差，让人想点进来
   - 好标题："AI写作最大的坑，不是写不好"
   - 坏标题："AI写作的优缺点分析"

2. 开头：用一个具体场景或问题切入
   - 好开头："上周有个读者私信我，说他用AI写了一篇文章，发出去被骂惨了。"
   - 坏开头："在当今数字化时代，AI写作已经成为..."

3. 正文：每段不超过3句话，多用短句
   - 好段落："说实话，这工具我用了三个月。最让我惊喜的是它的批量处理功能。以前要手动改一天的文档，它半小时就搞定了。"
   - 坏段落："该工具具有强大的功能特性，能够有效提升工作效率，其先进的算法和丰富的功能特性，使其成为内容创作者的理想选择。"

4. 结尾：留一个互动问题
   - 好结尾："你遇到过这种情况吗？欢迎在评论区分享你的经历。"
   - 坏结尾："综上所述，AI写作是一个值得探索的领域。"

5. 语言：像朋友聊天一样自然
   - 多用"你""我"
   - 多用口语化表达
   - 多用反问句
```

### 公众号去AI味检查清单
- [ ] 标题有没有悬念或反差？
- [ ] 开头是不是用具体场景切入？
- [ ] 每段是不是不超过3句话？
- [ ] 有没有"在当今""随着""综上"等万能表达？
- [ ] 结尾有没有互动问题？

## 📕 小红书去AI味技巧

### 小红书的特点
- 读者期待干货和颜值
- 读者喜欢短平快的内容
- 读者讨厌长篇大论
- 读者喜欢emoji和分点

### 小红书去AI味Prompt
```
帮我写一篇小红书笔记，要求：

1. 标题：要吸引眼球，带emoji
   - 好标题："🔥 3个去AI味技巧，让你的文章秒变人写的！"
   - 坏标题："去AI味的方法介绍"

2. 正文：分点列举，每点不超过2句话
   - 好格式：
     ✅ 技巧1：删除万能开头
     ✅ 技巧2：加入个人经历
     ✅ 技巧3：用口语化表达
   - 坏格式：大段文字堆砌

3. 语言：要接地气，像朋友分享
   - 好表达："姐妹们！这个方法真的绝了！"
   - 坏表达："该方法具有显著的效果。"

4. 结尾：引导互动
   - 好结尾："你们还有什么去AI味的好方法？评论区告诉我！"
   - 坏结尾："希望这些方法对你有所帮助。"

5. 标签：要精准，带热门标签
   - 好标签：#去AI味 #AI写作 #写作技巧 #自媒体干货
   - 坏标签：#AI #写作
```

### 小红书去AI味检查清单
- [ ] 标题有没有带emoji？
- [ ] 正文有没有分点列举？
- [ ] 每点是不是不超过2句话？
- [ ] 语言有没有接地气？
- [ ] 结尾有没有引导互动？
- [ ] 标签有没有精准且热门？

## 📝 知乎去AI味技巧

### 知乎的特点
- 读者期待专业分析
- 读者喜欢深度内容
- 读者讨厌营销和水文
- 读者喜欢有理有据的表达

### 知乎去AI味Prompt
```
帮我写一篇知乎回答，要求：

1. 开头：直接回答问题，不要绕弯子
   - 好开头："简单说：去AI味的核心是'注入人味'。"
   - 坏开头："关于这个问题，我想从多个角度来分析。"

2. 正文：要有理有据，用数据和案例支撑
   - 好表达："我测试了5个工具，只有2个能真正去AI味。"
   - 坏表达："该工具具有良好的效果。"

3. 语言：要专业但不学术
   - 好表达："说白了，去AI味就是让AI写得像人。"
   - 坏表达："去AI味是指通过特定方法消除人工智能生成内容的非自然特征。"

4. 结尾：给出明确建议
   - 好结尾："我的建议是：先学Prompt，再学改稿，最后学风格控制。"
   - 坏结尾："希望这些方法对你有所帮助。"

5. 格式：用加粗、列表、引用等格式增强可读性
```

### 知乎去AI味检查清单
- [ ] 开头有没有直接回答问题？
- [ ] 有没有用数据和案例支撑观点？
- [ ] 语言有没有专业但不学术？
- [ ] 结尾有没有给出明确建议？
- [ ] 有没有用格式增强可读性？

## 🎬 抖音/短视频去AI味技巧

### 抖音的特点
- 读者期待快节奏
- 读者喜欢抓眼球的内容
- 读者讨厌慢热和铺垫
- 读者喜欢刺激和冲突

### 抖音去AI味Prompt
```
帮我写一个抖音短视频脚本，要求：

1. 开头（前3秒）：必须抓眼球
   - 好开头："AI写出来的东西，一眼就能看出来是AI写的！"
   - 坏开头："大家好，今天我来分享一下AI写作的技巧。"

2. 正文（30秒内）：每句话不超过10个字
   - 好表达："去AI味，就三招。第一招：删废话。第二招：加人味。第三招：改风格。"
   - 坏表达："首先，我们需要了解AI写作的基本原理，然后学习如何撰写有效的Prompt..."

3. 结尾：留悬念或引导互动
   - 好结尾："想知道具体怎么操作？关注我，下期教你！"
   - 坏结尾："希望这个视频对你有所帮助。"

4. 语言：要口语化，像面对面聊天
   - 多用"你""我"
   - 多用短句
   - 多用反问句
```

### 抖音去AI味检查清单
- [ ] 开头前3秒有没有抓眼球？
- [ ] 每句话是不是不超过10个字？
- [ ] 结尾有没有留悬念或引导互动？
- [ ] 语言有没有口语化？

## 🎮 B站去AI味技巧

### B站的特点
- 读者期待有趣和有用
- 读者喜欢有梗的内容
- 读者讨厌枯燥和说教
- 读者喜欢UP主的个人风格

### B站去AI味Prompt
```
帮我写一个B站视频文案，要求：

1. 开头：要有梗，让人想继续看
   - 好开头："你知道吗？AI写出来的东西，就像方便面——闻着香，吃着没味。"
   - 坏开头："大家好，今天我们来聊聊AI写作。"

2. 正文：要有UP主的个人风格
   - 好表达："我试了10个去AI味的方法，只有3个真正有用。想知道是哪3个？继续看！"
   - 坏表达："去AI味的方法有很多种，下面我将为大家介绍..."

3. 结尾：要有互动和引导
   - 好结尾："你们还有什么去AI味的好方法？弹幕告诉我！别忘了三连！"
   - 坏结尾："希望这个视频对你有所帮助。"

4. 语言：要有UP主的个人风格
   - 多用"我""你"
   - 多用梗和段子
   - 多用反问和互动
```

### B站去AI味检查清单
- [ ] 开头有没有梗？
- [ ] 有没有UP主的个人风格？
- [ ] 结尾有没有互动和引导？
- [ ] 语言有没有有趣和有用？

## 🛠️ 通用去AI味工具推荐

### 工具一：AI去AI味工具

**推荐工具**：
- Claude（最擅长去AI味）
- ChatGPT（需要好的Prompt）
- DeepSeek（性价比高）

**使用方法**：
用前面介绍的Prompt模板，让AI帮你去AI味。

### 工具二：人工改稿

**推荐方法**：
1. 先让AI生成初稿
2. 用去AI味Prompt让AI自己改一遍
3. 你自己再改一遍
4. 大声读一遍，不顺的地方再改

### 工具三：风格参考

**推荐方法**：
1. 找一个你喜欢的作者
2. 分析他的写作风格
3. 让AI模仿他的风格写作
4. 你自己再调整
