## 📌 一句话总结

**识别AI味的第一步，是知道哪些词是AI的"身份证"。**

---

## 🎯 为什么要建立AI高频词词典？

AI写作有明显的"词汇指纹"——某些词和句式出现的频率远高于人类写作。掌握这些高频词，你就能：

1. **快速识别**：一眼看出文章是不是AI写的
2. **精准替换**：知道该把哪些词换成什么
3. **预防为主**：在 Prompt 里直接禁止这些词

---

## 🔴 万能开头词（必须删除）

| AI高频开头 | 替换方案 | 说明 |
|-----------|---------|------|
| 在当今数字化时代 | 最近/如今/你有没有发现 | 直接切入主题 |
| 随着...的快速发展 | 删掉，直接说事 | 不需要铺垫 |
| 值得一提的是 | 说个有意思的/重点是 | 口语化 |
| 不得不承认 | 说实话/其实 | 更自然 |
| 毫无疑问 | 删掉，直接说观点 | 不需要强调 |
| 众所周知 | 你知道吗/大家都说 | 更亲切 |
| 在...方面 | 删掉，直接说 | 去掉介词结构 |
| 从某种意义上来说 | 删掉，直接表态 | 不要模棱两可 |

---

## 🟡 万能过渡词（必须替换）

| AI高频过渡 | 替换方案 | 说明 |
|-----------|---------|------|
| 然而 | 但是/不过/但 | 更口语 |
| 此外 | 另外/还有/对了 | 更自然 |
| 因此 | 所以/这么一来 | 更口语 |
| 综上所述 | 总之/说白了/一句话总结 | 不要官方腔 |
| 总而言之 | 删掉或用"说到底" | 不要重复总结 |
| 不仅...而且... | 不光...还... | 更口语 |
| 一方面...另一方面... | 其实...但也有... | 拆掉八股结构 |
| 换句话说 | 说白了/简单说 | 更直接 |
| 具体来说 | 比如/举个例子 | 更具体 |
| 与此同时 | 这时候/同时 | 更简洁 |

---

## 🟠 空洞形容词（必须量化）

| AI高频形容词 | 替换方案 | 说明 |
|------------|---------|------|
| 强大的 | 用具体功能替代 | "强大的功能"→"支持批量处理" |
| 高效的 | 用数据替代 | "高效"→"30分钟搞定以前1天的活" |
| 优质的 | 用具体效果替代 | "优质内容"→"阅读量破万的文章" |
| 丰富的 | 列出具体项目 | "丰富功能"→"支持10种格式导出" |
| 先进的 | 用技术细节替代 | "先进算法"→"基于GPT-4的推理" |
| 显著的 | 用数据替代 | "显著提升"→"效率提升3倍" |
| 深入的 | 用具体内容替代 | "深入分析"→"逐句拆解了50篇文章" |
| 全面的 | 列出覆盖范围 | "全面覆盖"→"涵盖公众号/小红书/知乎" |

## 🔵 AI味句式（必须拆解）

| AI味句式 | 问题 | 改写方式 |
|---------|------|---------|
| 具有...的能力 | 被动、官方 | 直接说"能..." |
| 使其成为...的理想选择 | 空洞推荐 | "我推荐它因为..." |
| 在...方面发挥着重要作用 | 模糊 | 直接说做了什么 |
| 为...提供了有力支撑 | 官方腔 | "帮了大忙" |
| 值得我们深入探讨 | 废话 | 直接开始探讨 |
| 这无疑是一个值得关注的话题 | 废话 | 删掉 |
| 让我们一起来看看 | AI口头禅 | 直接看 |
| 接下来，我们将从几个方面来分析 | 八股预告 | 直接开始分析 |

---

## 🟣 AI味结尾（必须重写）

| AI味结尾 | 替换方案 | 说明 |
|---------|---------|------|
| 综上所述 | 删掉或用"一句话" | 不要总结腔 |
| 希望这篇文章对你有帮助 | 你遇到过这种情况吗？ | 互动而非客套 |
| 让我们一起期待未来 | 删掉 | 不要空洞展望 |
| 未来，...将继续发展 | 删掉或给具体预测 | 不要套话 |
| 感谢阅读 | 你还有什么问题？ | 引导互动 |
| 以上就是今天的分享 | 删掉 | 不要宣告结束 |
| 如果你觉得有用，请点赞收藏 | 你们觉得呢？ | 更自然的引导 |

## 🧪 实战练习

拿你最近写的一篇文章，用这个词典做一次"体检"：

1. 搜索所有"在当今""随着""综上"→删除或替换
2. 搜索所有"然而""此外""因此"→换成口语词
3. 搜索所有"强大的""高效的""优质的"→量化或删除
4. 搜索所有"具有...的能力"→改成主动句
5. 读一遍，还有哪里像AI写的？→继续改

## 💡 进阶技巧：在 Prompt 里禁词

在写 Prompt 时直接加入禁词列表：

```
请帮我写一篇文章，禁止使用以下词汇：
- 在当今、随着、值得一提、不得不承认
- 然而、此外、因此、综上所述
- 强大的、高效的、优质的、丰富的
- 具有...的能力、使其成为...的理想选择
- 希望对你有帮助、感谢阅读
```
