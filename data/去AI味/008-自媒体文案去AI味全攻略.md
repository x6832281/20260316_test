## 🎯 自媒体文案AI味的特征

自媒体读者对AI味特别敏感，因为他们每天刷大量内容，已经形成了"AI味雷达"。

### 公众号AI味特征
- 标题太"正经"：没有悬念、没有反差、没有数字
- 开头太"官方"：在当今、随着、值得一提
- 结尾太"客套"：希望对你有帮助、感谢阅读
- 段落太"整齐"：每段都是3句话，像列队

### 小红书AI味特征
- 标题没有emoji：小红书标题必须有emoji
- 正文没有分点：小红书必须分点列举
- 语言太"书面"：小红书要像朋友聊天
- 标签太"通用"：#AI #写作 → 要精准

### 知乎AI味特征
- 开头太"绕弯"：知乎要直接回答问题
- 论证太"空洞"：知乎要数据和案例支撑
- 结尾太"模糊"：知乎要给明确建议
- 格式太"朴素"：知乎要用加粗、列表、引用

## 📱 公众号去AI味实战

### 标题去AI味

**AI味标题**：
- "AI写作的优缺点分析"
- "如何提高AI写作效率"
- "AI写作工具推荐"

**人味标题**：
- "AI写作最大的坑，不是写不好"
- "我用AI写了100篇文章，发现了一个秘密"
- "这3个AI工具，让我效率翻了5倍"

**公式**：悬念/反差 + 数字 + 具体利益

### 开头去AI味

**AI味开头**：
```
在当今数字化时代，AI写作已经成为内容创作领域的重要工具。随着大语言模型的快速发展，越来越多的人开始使用AI辅助写作。
```

**人味开头**：
```
上周有个读者私信我，说他用AI写了一篇文章，发出去被骂惨了。

为什么？因为一眼就能看出是AI写的。
```

**公式**：具体场景/问题 + 冲突/悬念

### 正文去AI味

**AI味正文**：
```
AI写作具有以下优点：首先，它可以提高写作效率；其次，它可以提供创作灵感；最后，它可以帮助用户克服写作障碍。然而，AI写作也存在一些缺点，如缺乏原创性、容易产生AI味等。
```

**人味正文**：
```
说实话，AI写作就是个"效率工具"。

我试过用AI写公众号文章，效率确实高了——以前要写3小时的文章，现在30分钟就搞定。

但质量嘛...怎么说呢，就像吃快餐——能填饱肚子，但别指望它有营养。
```

**公式**：个人观点 + 具体案例 + 口语化表达

## 📕 小红书去AI味实战

### 标题去AI味

**AI味标题**：
- "去AI味的方法介绍"
- "AI写作技巧分享"
- "如何让AI写作更自然"

**人味标题**：
- "🔥 3个去AI味技巧，让你的文章秒变人写的！"
- "姐妹们！这个AI工具真的绝了！"
- "💡 AI写作小白必看！这些坑我都帮你踩过了"

**公式**：emoji + 数字 + 情感词 + 具体利益

### 正文去AI味

**AI味正文**：
```
去AI味的方法有很多种，包括修改开头结尾、调整句式结构、加入个人经历等。这些方法可以有效提升文章的自然度。
```

**人味正文**：
```
✅ 技巧1：删除万能开头
"在当今数字化时代"→ 直接说重点

✅ 技巧2：加入个人经历
"我试过..." → 让读者觉得你是真人

✅ 技巧3：用口语化表达
"说白了" → 比"综上所述"强100倍
```

**公式**：分点列举 + emoji + 口语化 + 具体操作

## 📝 知乎去AI味实战

### 开头去AI味

**AI味开头**：
```
关于这个问题，我想从多个角度来分析。首先，我们需要了解AI写作的基本原理...
```

**人味开头**：
```
简单说：去AI味的核心是"注入人味"。

我测了10个去AI味的方法，只有3个真正有用。想知道是哪3个？继续看。
```

**公式**：直接回答 + 数据支撑 + 悬念引导

### 正文去AI味

**AI味正文**：
```
AI写作的优势在于效率高、成本低、可批量生产。但其劣势也很明显：缺乏原创性、容易产生AI味、难以体现个人风格。
```

**人味正文**：
```
**先说结论**：AI写作适合打草稿，不适合直接发布。

我测试了5个AI写作工具，发现：
- 效率确实高了3倍（数据来源：我自己写100篇文章的对比）
- 但AI味检测率高达85%（用GPTZero测试）
- 改稿时间占了总时间的60%

**所以我的建议是**：让AI打草稿，你来改。
```

**公式**：加粗重点 + 数据支撑 + 个人建议 + 列表格式

## 🔧 自媒体去AI味工具箱

### 工具一：AI去AI味
用Claude或ChatGPT，直接给去AI味Prompt。

### 工具二：人工改稿
先让AI写初稿 → 用去AI味Prompt改一遍 → 你自己再改一遍 → 大声读一遍。

### 工具三：风格参考
找一个你喜欢的作者，分析他的风格，让AI模仿。

### 工具四：读者测试
发给朋友看，问他们"觉得像AI写的吗"。

## ✅ 自媒体去AI味检查清单

### 公众号检查
- [ ] 标题有没有悬念或反差？
- [ ] 开头是不是用具体场景切入？
- [ ] 每段是不是不超过3句话？
- [ ] 有没有"在当今""随着""综上"等万能表达？
- [ ] 结尾有没有互动问题？

### 小红书检查
- [ ] 标题有没有带emoji？
- [ ] 正文有没有分点列举？
- [ ] 每点是不是不超过2句话？
- [ ] 语言有没有接地气？
- [ ] 结尾有没有引导互动？
- [ ] 标签有没有精准且热门？

### 知乎检查
- [ ] 开头有没有直接回答问题？
- [ ] 有没有用数据和案例支撑观点？
- [ ] 语言有没有专业但不学术？
- [ ] 结尾有没有给出明确建议？
- [ ] 有没有用格式增强可读性？
