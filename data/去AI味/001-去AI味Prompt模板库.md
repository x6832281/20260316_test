## 📌 一句话总结

**好的 Prompt 不是让 AI 写得更好，而是让 AI 写得更像人。**

---

## 🎯 为什么需要去AI味 Prompt？

直接让 AI 写文章，99% 会写出"在当今数字化时代""综上所述""值得注意的是"这类一眼AI的表达。

去AI味 Prompt 的核心思路：**不是让 AI 创作，而是让 AI 模仿人类的写作习惯。**

---

## 📝 万能去AI味 Prompt（通用版）

```
请帮我改写以下内容，要求：

1. 删除所有万能开头（"在当今""随着""值得一提的是"）
2. 删除所有万能结尾（"综上所述""总之""希望对你有帮助"）
3. 把长句拆成短句，每句话不超过20个字
4. 加入口语化表达（"说白了""其实""你想想"）
5. 加入个人观点和态度（"我觉得""说实话""我的经验是"）
6. 用具体例子替代抽象描述
7. 像朋友聊天一样自然

原文：
[粘贴你的内容]
```

## 📱 分平台 Prompt 模板

### 公众号去AI味 Prompt

```
帮我写一篇公众号文章，主题是[xxx]，要求：

标题：要有悬念或反差，让人想点进来
- 好标题示例："AI写作最大的坑，不是写不好"
- 坏标题示例："AI写作的优缺点分析"

开头：用一个具体场景或问题切入，不要用"在当今数字化时代"
- 好开头示例："上周有个读者私信我，说他用AI写了一篇文章，发出去被骂惨了。"

正文：
- 每段不超过3句话
- 多用短句（10-15字）
- 多用"你""我"
- 多用反问句
- 每隔3-4段加一个具体案例或个人经历

结尾：留一个互动问题，不要用"综上所述"
```

### 小红书去AI味 Prompt

```
帮我写一篇小红书笔记，主题是[xxx]，要求：

标题：带emoji，有数字，有反差
- 示例："🔥 3个去AI味技巧，让你的文章秒变人写的！"

正文：
- 分点列举，每点不超过2句话
- 多用emoji（但不要过度）
- 语言要接地气，像朋友分享
- 用"姐妹们""兄弟们"等称呼

标签：#去AI味 #AI写作 #写作技巧 #自媒体干货
```

### 知乎去AI味 Prompt

```
帮我写一篇知乎回答，问题是[xxx]，要求：

开头：直接回答问题，不要绕弯子
- 好开头："简单说：去AI味的核心是'注入人味'。"
- 坏开头："关于这个问题，我想从多个角度来分析。"

正文：
- 要有理有据，用数据和案例支撑
- 语言要专业但不学术
- 用加粗、列表、引用等格式增强可读性

结尾：给出明确建议，不要用"希望对你有帮助"
```

### 短视频脚本去AI味 Prompt

```
帮我写一个短视频脚本，主题是[xxx]，要求：

开头（前3秒）：必须抓眼球
- 好开头："AI写出来的东西，一眼就能看出来是AI写的！"
- 坏开头："大家好，今天我来分享一下AI写作的技巧。"

正文（30秒内）：
- 每句话不超过10个字
- 节奏快，信息密度高
- 多用短句和排比

结尾：留悬念或引导互动
```

## 🔧 场景化 Prompt 模板

### 场景一：改写已有文章

```
请帮我改写以下文章，让它读起来像人写的而不是AI写的。

改写规则：
1. 删除所有"在当今""随着""综上所述"等万能表达
2. 把"一方面...另一方面..."改成"其实...但也有..."
3. 把"值得注意的是"改成直接说重点
4. 加入至少2处个人观点（"我觉得""我的经验是"）
5. 加入至少1个具体例子
6. 把每段控制在3句话以内

原文：
[粘贴内容]
```

### 场景二：从零写去AI味文章

```
请帮我写一篇关于[主题]的文章，要求读起来完全不像AI写的。

写作要求：
1. 用第一人称"我"来写
2. 开头用一个具体场景或问题切入
3. 每段不超过3句话
4. 多用短句（10-15字）
5. 加入个人经历或案例
6. 用口语化表达（"说白了""其实""你想想"）
7. 结尾留一个互动问题

字数要求：[xxx]字左右
```

### 场景三：批量去AI味

```
请帮我批量改写以下内容，每段都要去AI味。

改写检查清单：
- [ ] 删除万能开头结尾
- [ ] 长句拆短句
- [ ] 加入口语化表达
- [ ] 加入个人观点
- [ ] 用具体例子替代抽象描述
- [ ] 大声读一遍，不顺的地方再改

内容：
[粘贴多段内容]
```

## ✅ Prompt 使用检查清单

用完 Prompt 后，检查输出是否还有AI味：

- [ ] 有没有"在当今""随着""综上"等万能表达？
- [ ] 有没有"一方面...另一方面..."的八股结构？
- [ ] 有没有空洞的形容词堆砌？
- [ ] 读起来像朋友聊天还是像官方报告？
- [ ] 有没有具体的个人经历或案例？
