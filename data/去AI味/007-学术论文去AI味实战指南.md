## 📌 一句话总结

**学术论文的AI味最重——因为它天生就"官方"，去AI味反而更难。**

---

## 🎯 学术论文AI味的特殊性

学术论文本身就有"正式、客观、中立"的要求，这和AI的写作风格高度重叠。所以学术论文的AI味更隐蔽，但也更容易被检测器抓到。

### 学术论文AI味的三大特征

1. **句式太规整**：每句话都是"主语+谓语+宾语"的标准结构
2. **逻辑太清晰**：每个段落都是"论点+论据+结论"的完美格式
3. **用词太学术**：过度使用"研究表明""数据显示""综上所述"

---

## 🛠️ 学术论文去AI味四步法

### 第一步：打破句式规整性

**问题**：AI写的句子太整齐，像列队的士兵。

**解决**：
```
请在论文中加入以下句式变化：
1. 偶尔用倒装句
2. 偶尔用省略句
3. 偶尔用反问句
4. 偶尔用感叹句（谨慎使用）
5. 偶尔用破折号插入补充说明
```

**示例**：
- ❌ AI味："本研究采用问卷调查法收集数据。"
- ✅ 人味："数据怎么来的？问卷调查——发了500份，回收了423份。"

### 第二步：增加思维跳跃

**问题**：AI的逻辑太严密，每一步都"理所当然"。

**解决**：
```
请在论文中加入以下思维特征：
1. 偶尔跳过一些"显而易见"的推理步骤
2. 偶尔加入一些"不太确定"的表达
3. 偶尔加入一些"个人猜测"
4. 偶尔加入一些"有待验证"的观点
5. 偶尔加入一些"可能有争议"的判断
```

**示例**：
- ❌ AI味："根据统计分析，变量X与变量Y呈显著正相关（p<0.05）。"
- ✅ 人味："X和Y明显有关系——虽然我还不太确定是因果还是相关，但p值确实小于0.05。"

### 第三步：注入个人风格

**问题**：AI写的论文没有"作者感"，谁写都一样。

**解决**：
```
请在论文中加入以下个人特征：
1. 偶尔用第一人称"我"或"我们"
2. 偶尔加入一些个人经历或观察
3. 偶尔加入一些个人偏好或立场
4. 偶尔加入一些幽默或讽刺（谨慎使用）
5. 偶尔加入一些自嘲或谦虚
```

**示例**：
- ❌ AI味："本研究具有重要的理论意义和实践价值。"
- ✅ 人味："说实话，这篇论文的理论贡献有限，但对一线从业者可能有点用。"

### 第四步：打乱段落结构

**问题**：AI的段落太规整，每段都是"主题句+支撑句+总结句"。

**解决**：
```
请在论文中加入以下段落变化：
1. 偶尔用超短段落（只有1-2句话）
2. 偶尔用超长段落（超过5句话）
3. 偶尔用无主题句的段落
4. 偶尔用反问句开头的段落
5. 偶尔用引用或案例开头的段落
```

## 📝 学术论文去AI味 Prompt 模板

```
请帮我改写以下学术论文段落，使其更像人类学者写的：

改写规则：
1. 打破句式规整性——加入倒装、省略、反问等句式
2. 增加思维跳跃——跳过一些"显而易见"的推理步骤
3. 注入个人风格——偶尔用"我"或加入个人判断
4. 打乱段落结构——混合长短段落，打破"主题句+支撑句"的固定格式
5. 保留学术性——不能太口语化，要保持学术论文的基本规范

原文：
[粘贴论文段落]
```

---

## ⚠️ 学术论文去AI味的注意事项

1. **不能过度口语化**：学术论文需要保持一定的正式性
2. **不能丢失准确性**：去AI味不能牺牲内容的准确性
3. **不能破坏逻辑**：去AI味不能让论文变得逻辑混乱
4. **不能违反学术规范**：引用、数据、结论必须准确
5. **保留专业术语**：专业术语不能随意替换
