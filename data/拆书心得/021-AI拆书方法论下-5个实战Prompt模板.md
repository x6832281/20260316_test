## 📌 一句话总结

**5个从"选书"到"写拆书笔记"的实战Prompt模板，复制粘贴即可使用——让AI帮你把任何一本经典拆成写作课。**

---

## 🔧 模板一：选书——这本书值得拆吗？

不是每本书都值得拆。拆书要选"技法密度高"的书——每一段都有东西可学的书。

```
我正在考虑拆解[书名]这本书。请从以下角度帮我评估它是否适合拆书：

1. 这本书的文学地位如何？它为什么被认为是经典？
2. 这本书最突出的写作特点是什么？（叙事结构、语言风格、人物塑造等）
3. 这本书中有哪些段落是"写作课级别"的？（举例3-5个经典段落）
4. 对于想学写作的新人来说，这本书最值得学的是什么？
5. 拆解这本书，你建议聚焦哪些主题？（给出5-8个拆书主题建议）

请用中文回答，每个问题给出具体、有深度的分析，不要泛泛而谈。
```

**使用场景**：当你不确定某本书值不值得花时间拆时，先用这个模板让AI帮你评估。

---

## 🔧 模板二：逐段精拆——深度分析一个段落

这是最核心的模板。每次只喂一个段落（200-500字），让AI做深度分析。

```
请对以下文学段落进行深度拆解分析：

[粘贴段落原文，200-500字]

分析要求：
1. **字面还原**：这段话写了什么？用一句话概括。
2. **句式分析**：这段话的句式有什么特点？（长短、节奏、重复、倒装等）这种句式产生了什么效果？
3. **意象解读**：这段话用了哪些意象？每个意象承载了什么含义？意象之间有什么关系？
4. **叙事视角**：这段话是谁在看？看到了什么？没看到什么？这个视角有什么特殊效果？
5. **情感层次**：这段话的表面情感是什么？深层情感是什么？两者之间有什么张力？
6. **技法提炼**：从这段话中可以提炼出哪2-3条可操作的写作技法？每条技法给出"错误示范"和"正确示范"。
7. **写作练习**：基于以上分析，设计一个200字的写作练习题。

请用中文回答，分析要具体、深入，引用原文中的具体词句来支撑你的分析。不要泛泛而谈，不要堆砌文学术语。
```

**使用场景**：这是你用得最多的模板。每拆一段，就用一次。

## 🔧 模板三：跨段落对比——同一个主题在不同段落中的写法

```
请对比分析以下两个段落的写法异同：

段落A（出自[书名1]）：
[粘贴段落原文]

段落B（出自[书名2]）：
[粘贴段落原文]

对比要求：
1. 两个段落写的主题有什么共同点？
2. 两个段落在写法上有什么关键差异？（句式、意象、视角、情感等）
3. 这些差异反映了两位作者怎样的文学观/人生观？
4. 如果你要写一个类似主题的段落，你会选择哪种写法？为什么？
5. 请用两种不同的写法各写一个100字的示范段落。

请用中文回答，对比要具体、有洞察力，不要只说"A用了比喻B用了排比"这种表面分析。
```

**使用场景**：当你想深入理解"同一个主题可以有多少种写法"时使用。比如对比萧红和鲁迅写童年、对比余华和莫言写死亡。

## 🔧 模板四：人物弧光——一个人物的变化轨迹

```
请分析[书名]中[人物名]这个人物的变化轨迹：

1. **起点**：这个人物出场时是什么状态？（性格、处境、信念）
2. **转折点**：哪些事件改变了这个人物？（列出3-5个关键事件）
3. **终点**：这个人物最终变成了什么样？
4. **变化逻辑**：从起点到终点，变化的内在逻辑是什么？（不是"发生了什么"，而是"为什么会这样变化"）
5. **作者态度**：作者对这个人物的态度是什么？同情？批判？复杂？
6. **写作启示**：从这个人物的变化轨迹中，可以学到什么关于"人物塑造"的技法？

请用中文回答，引用原文中的具体情节和对话来支撑你的分析。
```

**使用场景**：当你想理解"怎么写一个有深度的人物"时使用。人物弧光是长篇小说最核心的技法之一。

## 🔧 模板五：生成拆书笔记——把分析变成可发布的文章

```
基于以下拆书分析，请生成一篇拆书笔记文章：

[粘贴之前的拆书分析结果]

文章要求：
1. **标题**：[书名]·[段落主题]：[一句话核心洞察]
2. **一句话总结**：用一句有力的话概括这段拆书的核心发现
3. **拆书正文**：2-4个小节，每节有原文引用+分析+洞察，小节标题要有吸引力
4. **写作技法提炼**：3条可操作的技法，每条有"技法名+一句话说明+错误示范+正确示范"
5. **写作练习**：一个200字的实操练习题
6. **一句话记住**：用一句金句总结这篇拆书

风格要求：
- 语言要生动有力，不要学术腔
- 多用短句，少用长句
- 引用原文时用引用格式
- 每个小节要有"洞察"，不要只做表面分析
- 字数控制在1500-2000字
```

**使用场景**：当你完成了深度分析，想把它整理成一篇可以发布/分享的拆书笔记时使用。

---

## 💡 使用建议

1. **按顺序使用**：先模板一选书→模板二逐段拆→模板三/四深化→模板五成文
2. **每次只拆一段**：不要贪多，200-500字一段，拆深拆透
3. **AI的分析要批判性地看**：AI有时会"过度解读"或"说正确的废话"，你要用自己的判断力去筛选
4. **保存你的Prompt**：这5个模板可以根据不同书籍微调，形成你自己的拆书工具箱

---

## 🔥 拆书的终极心法

> 拆书的目的不是"读懂一本书"，而是"从一本书中学到一种写法"。每拆一段，你都应该能回答：**我学到了什么？我能怎么用？**
