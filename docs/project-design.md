# AI 萌新小窝 - 项目整体设计文档

## 项目概述

**AI 萌新小窝** 是面向完全不懂编程和AI的新人，专注于AI文学创作技巧的科普网站。

### 目标用户
- 对AI文学创作感兴趣的创作者
- 想学习用AI写小说、文案、段子的新手
- 希望通过AI提升写作效率的人群

### 核心价值
- 提供通俗易懂的AI文学创作教程
- 建立活跃的AI创作者社区（长期目标）

## 技术架构

### 前端技术
- HTML5 + CSS3 + JavaScript（纯静态）
- 响应式设计，适配桌面和移动设备
- Markdown 渲染（marked.js）

### 数据存储
- 内容：Markdown 文件存储在 `data/` 目录
- 留言：localStorage（轻量级，无后端）

### 内容生成
- Skill 系统：`.trae/skills/` 目录下定义内容生成规则
- 每次生成6篇文章，保持格式统一

---

## 交互设计

### 创作者交流区（留言板升级版）
- 留言类型选择：问题咨询、作品分享、经验交流、建议反馈
- 不同类型用颜色标签区分
- 作品分享类型支持 Markdown 预览

### AI创作标签页
- 分类筛选按钮：全部、小说、文案、段子
- 文章卡片显示类型标签
- 底部引导创作者通过交流区分享作品
