核心游戏引擎 (必须选一个深入)

🤖
Godot Engine

完全免费开源！节点式架构极度优雅，使用类似 Python 的 GDScript，近年最火引擎。
📚
Godot 官方文档

官方中文文档，包含完整的 API 参考和教程。
🛒
Godot 资源库

官方资源库，包含大量免费的插件、脚本和素材。
🌍
Godot 社区

官方论坛和社区，获取最新资讯和技术支持。
🐙
Godot GitHub

Godot 源码仓库，参与开源贡献。
💬
Godot Forums

Godot 社区论坛，提问和分享经验。
🧊
Unity

最适合新手的 2D/3D 引擎，C# 编程，社区教程全网最多，独立游戏首选。
🔥
Unreal Engine (UE5)

画面表现天花板，强大的蓝图(Blueprint)系统让你可以不写代码做游戏。
🐹
GameMaker

2D 游戏的王者，《传说之下》《蔚蓝》同款引擎，适合纯 2D 像素风开发。

代码与系统框架 (不用重复造轮子)

💻
浅墨的 GitHub

已故大神浅墨留下的宝贵遗产，包含大量 Unity 渲染和架构的优质代码。

国内极具影响力的 Unity 开源框架，包含 UI、资源管理、网络等完整模块。

PlayMaker (Unity)

《空洞骑士》同款状态机可视化编程插件，不写代码也能做复杂逻辑。

叙事与关卡设计 (让游戏变好玩)

🕸️
Twine

开源的互动文字游戏制作工具，非常适合用来做游戏剧情的网状叙事设计。
✒️
Ink (Inkle Studios)

强大的叙事脚本语言，被广泛用于《黑帝斯》、《极乐迪斯科》等神作的对话系统。
🗺️
Tiled

全球最流行的免费 2D 瓦片地图编辑器，支持导出给几乎所有游戏引擎使用。
🧠
Ludum Dare (历届作品)

全球最大的 Game Jam，去玩历届冠军作品是学习关卡设计和核心玩法的最快途径。
