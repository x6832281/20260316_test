# 拆书心得 —— 逐段拆解经典，学写作内功

## 模块定位

**拆书心得** 是 AI 萌新小窝的六大核心模块之一，位于首页第六栏。这是本平台 **最重要的核心模块**（第一优先级），体现了"用 AI 教新人读书和写作"的初心。

> **一句话定位**：让 AI 像一个文学导师，带着你逐字逐段地品味经典，每一句话、每一个用词都细细拆解——不是概括，是品味；不是速读，是精读。

### 模块目标
- 用 AI 逐段拆解经典名著的写作技法（叙事结构、人物塑造、场景描写、语言风格）
- 让零基础新人直观看到"好文字长什么样"
- 从经典中提炼可操作的写作技法，直接应用到 AI 辅助创作中
- 建立"拆书实践 → 文学理论 → AI 写作"的学习闭环

### 核心理念
> 拆书不是总结，是品味。写好小说的前提是读好小说。每一段拆解都是一个小型的"写作大师课"。

## 拆书方法论

### 核心方法：逐句拆解法

不是概括一段话的大意，而是对每一句话都追问三个问题：

| 问题 | 追问方向 | 示例（以萧红为例） |
|------|----------|-------------------|
| **写了什么？** | 这句话描述了什么事/物/人？ | "蝴蝶有白蝴蝶、黄蝴蝶" |
| **怎么写的？** | 用了什么句式/修辞/视角？ | 短句堆叠，模仿儿童数家珍的说话节奏 |
| **为什么这样写？** | 换一种写法会怎样？作者的意图是什么？ | 用孩童视角过滤成人世界的残酷，以乐写哀 |

### 拆书分析框架

每段拆解遵循以下分析框架：

```markdown
### [段落主题小标题]

> [引用原文段落]

#### 技法拆解

**[技法名称]：[具体分析]**

- **写了什么**：[对原文内容的客观描述]
- **怎么写的**：[句式特征、修辞手法、节奏控制的专业分析]
- **为什么这样写**：[作者的深层意图 + 换一种写法会损失什么]

#### 💡 写作启示

[从这一段的拆解中可以学到什么，如何应用到自己的 AI 写作中]
```

### 必拆维度清单

每段原文至少从以下维度中选 3 个进行深入分析：

| 维度 | 分析指标 | 权重 |
|------|----------|------|
| **句式节奏** | 长句 vs 短句的比例、切换节点 | 必选 ⭐⭐⭐ |
| **用词选择** | 动词/形容词的"精准度"，是否能被替换 | 必选 ⭐⭐⭐ |
| **修辞手法** | 比喻、排比、对比、反讽等 | 推荐 ⭐⭐ |
| **情感控制** | 情感的强弱与克制，情感爆发的"时机" | 推荐 ⭐⭐ |
| **留白艺术** | 哪些信息"没写"？留白产生了什么效果？ | 推荐 ⭐⭐ |
| **视角切换** | 全知/限制/客观视角的切换节点 | 可选 ⭐ |
| **时空处理** | 时间的压缩与扩张，场景的切换方式 | 可选 ⭐ |

## 写作规范

### 语言风格
- **学术深度 + 课堂亲密度**：像一个特别会讲课的文学老师在带你逐句读
- 每一段拆解都要回答三个问题：写了什么、怎么写的、为什么这样写
- 引用原文后必须紧跟"拆解分析"（不能引用完就跳到下一段）
- 多使用对比手法："如果作者写成了XX会怎样？"——帮助理解"为什么这样写是好的"

### 内容要求
- 每篇不少于 800 字（引用原文不计入字数）
- 至少提炼 2 个可操作的写作技法
- 必须提供 1 个以上可直接使用的 AI 写作联动 Prompt
- 至少包含 3 段以上的逐句/逐段拆解

### 拆书深度标准
| 层次 | 标准 | 要求 |
|------|------|------|
| **浅层** | "这段话写了园子里的景物" | ❌ 不合格 |
| **中层** | "用了大量的短句，营造轻快的节奏" | ✅ 合格 |
| **深层** | "短句节奏模仿了儿童语言，这层模仿本身就是一种叙事策略——用童稚的形式过滤成人世界的残酷" | ⭐ 优秀 |

### 禁止事项
- 禁止将拆书变成"段落大意概括"（这是本模块最大的"红线"）
- 禁止大段引用原文而不做分析（引用与分析的合理比例约 1:3）
- 禁止使用空洞的评价——"写得太好了""暗示了深刻的含义"
- 禁止用 AI 直接生成拆书内容（拆书的核心价值在于"人的品味"，AI 可以辅助但不能替代）

## 使用方式

当用户需要：
1. **学写作技法**：逐段拆解某部经典的精彩段落
2. **系统学拆书方法**：生成拆书方法论和 Prompt 模板
3. **做写作练习**：生成"拆书→AI 写作联动"的练习 Prompt
4. **选书参考**：按"写作技法学习价值"推荐可拆解的书目

## 注意事项

- **引用版权**：引用原文需注意版权合规——优先选择进入公共领域的作品（如中国现代作家的公版小说、古典名著等）
- **拆书不是"书评"**：拆书聚焦在"写作技法"的分析上，不做文学评价（不说"这本书很好/很差"）
- **Prompt 必须原创**：AI 写作联动 Prompt 是基于本期拆析内容"定制"的，不能是通用的模板化 Prompt
- **一期只拆一段**：不贪多——宁可一期只拆一小段拆深了，不能一期拆一堆都只点到为止
- **新人友好**：即使读者没有读过全书，读完这篇拆书心得也能独立欣赏这段文字的写作价值
