# 科技热搜抓取工具

这个技能用于抓取网络上的科技热搜信息，整理成排行文档并保存到项目的热搜排行文件夹。

## 功能说明

- **抓取来源**：36氪、虎嗅网、IT之家、机器之心、雷锋网、快科技、极客公园、TechCrunch、The Verge、MIT Technology Review、Wired、CSDN、X-TECHCON 科技区角
- **抓取内容**：当前最热门的10个科技信息
- **输出格式**：Markdown文档，保存到 `data/热搜排行/` 目录
- **文件名格式**：`hot-search-YYYY-MM-DD.md`

## 执行逻辑

1. **数据抓取**：从各个科技网站获取热搜信息
2. **数据处理**：提取标题、链接、来源、热度、摘要等信息
3. **排序处理**：根据热度值对信息进行排序，取前10条
4. **生成文档**：按照指定格式生成Markdown文档
5. **保存文件**：保存到 `data/热搜排行/` 目录

## 错误处理

- **网络请求失败**：自动使用预设的模拟数据
- **URL验证失败**：在相关链接后显示警告提示
- **文件保存失败**：输出错误信息并尝试重试

## 使用方式

当用户请求执行科技热搜抓取时：
1. 调用此skill
2. 执行数据抓取和处理
3. 生成并保存热搜排行文档
4. 告知用户文档已生成完成

## 注意事项

- 抓取时会自动跳过无法访问的网站
- 每个网站请求间隔1秒，避免被封禁
- 生成的文档会自动保存到项目根目录的 `data/热搜排行/` 下
