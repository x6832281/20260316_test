# 科技热搜抓取工具

这个技能用于抓取网络上的科技热搜信息，整理成排行文档并保存到项目的热搜排行文件夹。

## 功能说明

- **抓取来源**：新浪科技、腾讯科技、网易科技、凤凰科技、IT之家、中关村在线、太平洋科技、天极网、泡泡网
- **抓取内容**：当前最热门的10个科技信息
- **输出格式**：Markdown文档，保存到 `data/热搜排行/` 目录
- **文件名格式**：`hot-search-latest.md`（固定文件名，只保留一份最新文档）

## 执行逻辑

1. **数据抓取**：从各个科技网站获取热搜信息
2. **数据处理**：提取标题、链接、来源、热度、摘要等信息
3. **排序处理**：根据热度值对信息进行排序，取前10条
4. **生成文档**：按照指定格式生成Markdown文档
5. **保存文件**：删除旧文件后保存到 `data/热搜排行/hot-search-latest.md`

## 重要规则

- **只保留一份最新文档**：文件名固定为 `hot-search-latest.md`，每次生成前删除旧文件
- **详细解读不与一句话总结重复**：一句话总结是高度概括，详细解读要展开技术细节、行业背景和数据
- **不显示"暂无评价"**：缺点、适合人群、使用难度、竞品对比等字段只在有实际内容时才显示
- **首页渲染**：首页通过 `render-content.js` 中的 `getLatestTrendingData()` 函数读取 `data/热搜排行/hot-search-latest.md` 文件来渲染热搜排行

## 错误处理

- **网络请求失败**：自动使用预设的模拟数据
- **URL验证失败**：在相关链接后显示警告提示
- **文件保存失败**：输出错误信息并尝试重试

## 使用方式

当用户请求执行科技热搜抓取时：
1. 调用此skill
2. 执行数据抓取和处理
3. 生成并保存热搜排行文档（固定文件名 `hot-search-latest.md`）
4. 告知用户文档已生成完成

## 注意事项

- 抓取时会自动跳过无法访问的网站
- 每个网站请求间隔1秒，避免被封禁
- 生成的文档固定保存为 `data/热搜排行/hot-search-latest.md`，确保只有一份最新文档
- 首页通过读取此文件来渲染热搜排行
