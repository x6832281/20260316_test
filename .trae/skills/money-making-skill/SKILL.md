# 搞钱创业项目抓取工具

这个技能用于抓取创业项目和收入信息，生成6篇中文总结文章并保存到项目的搞钱创业文件夹。

## 功能说明

- **抓取来源**：TrustMRR、Indie Hackers、IT 桔子、36 氪创投
- **抓取内容**：真实的创业项目和收入信息
- **输出格式**：Markdown文档，保存到 `data/搞钱创业/` 目录
- **生成数量**：每次生成6篇中文总结文章

## 生成格式

每个项目的Markdown格式：

```markdown
# [项目名称]

**💰 月收入**：XXX | **📅 更新日期**：YYYY-MM-DD

---

## 📝 一句话总结

[项目的一句话总结]

## 📋 详细介绍

[项目的详细介绍，包括功能、特点、商业模式等]

## 🎯 个人评价

[个人评价，包括项目的优缺点、市场前景、适合人群等]

## 🔧 技术栈

- [技术1]
- [技术2]
- [技术3]

## 🔗 相关链接

- [项目官网](链接)
- [产品介绍](链接，如果有)
- [收入证明](链接，如果有)

---

**© 2026 AI Tools Magazine | AI 萌新小窝 出品**

[返回首页](/)
```

## 执行逻辑

1. **数据抓取**：从指定网站获取创业项目和收入信息
2. **数据处理**：提取项目名称、描述、收入、商业模式、技术栈等信息
3. **排序筛选**：根据收入和更新时间筛选出最有价值的项目
4. **生成文档**：为每个项目生成中文总结文章
5. **保存文件**：保存到 `data/搞钱创业/` 目录
6. **渲染到首页**：通过前端JavaScript自动读取 `data/搞钱创业/` 目录下的文件并渲染到首页的搞钱排行部分

## 错误处理

- **网络请求失败**：自动使用预设的模拟数据
- **API限流**：添加请求间隔，避免被网站限流
- **文件保存失败**：输出错误信息并尝试重试

## 使用方式

当用户请求执行搞钱创业项目抓取时：
1. 调用此skill
2. 执行数据抓取和处理
3. 生成并保存项目总结文章
4. 告知用户文档已生成完成

## 文件结构

```
.trae/
  skills/
    money-making-skill/
      SKILL.md          # 技能配置文件
      money_making_crawler.py  # 搞钱创业项目抓取主程序
      run.py            # 执行脚本
```

## 注意事项

- 抓取时会自动跳过无法访问的项目
- 每个请求间隔2秒，避免被网站限流
- 生成的文档会自动保存到项目根目录的 `data/搞钱创业/` 下
- 优先抓取有真实收入数据的项目
- 确保项目信息的真实性和可靠性
