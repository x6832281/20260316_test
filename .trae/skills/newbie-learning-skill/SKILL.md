# 萌新学习内容生成工具 v2

这个技能用于为AI萌新生成聚焦**AI写作领域**的学习内容：

1. **GitHub 热门 AI 写作项目**：关注 inkOS、AI-Writer、NovelAI 等热门开源项目
2. **AI 写作 SKILL**：长篇连贯性写作、人设卡、大纲结构化、提示词工程
3. **主流博主教程**：按重要性排列——
   - 数字生命卡兹克（学心法/去AI味）
   - 秋芝（找工具/避坑）
   - 宝玉（调教AI/提示词）
   - 林亦 LYi（找灵感/脑洞）
   - 武彬（商业逻辑/人设）
   - 漫士沉思录（世界观/设定）
   - 汗青 AITalk（行业资讯）
   - 陈行之/皮特（故事内核）
   - 张然冉（视觉辅助）
   - 立党老师（素材搜集）

## 生成格式

每篇文档包含：
1. 标题（星标/流量/更新日期可选）
2. 📌 一句话总结（用于首页卡片展示）
3. 详细解释
4. 验证可访问的原文链接

## 输出

- 输出到 `data/萌新学习/` 目录
- 共 6 篇文档：1 篇 GitHub 项目 + 1 篇 SKILL + 4 篇博主教程合集
