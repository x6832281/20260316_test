# 萌新学习内容生成工具

这个技能用于为AI萌新生成学习内容，涵盖两大方向：
1. **AI/科技圈入门**：基础知识、术语、教程、视频、演讲、工具使用
2. **AI文学创作**：AI创作案例、技巧、相关网站、教程、写作基础、常用skill

## 功能说明

- **内容方向一**：AI/科技圈入门知识
  - AI基本术语通俗解释
  - 零基础AI入门教程推荐
  - B站热门AI学习视频
  - AI名人演讲精选
  - AI工具使用入门指南
  - AI效率提升技巧

- **内容方向二**：AI文学创作
  - AI写作入门：从零开始用AI写文章
  - AI小说创作：如何用AI辅助写小说
  - AI诗歌创作：用AI激发诗意灵感
  - AI文案写作：爆款文案的AI生成技巧
  - AI文学创作网站与工具推荐
  - AI写作常用Prompt技巧大全

- **输出格式**：Markdown文档，保存到 `data/萌新学习/` 目录
- **生成数量**：每次生成12篇学习文档（6篇入门 + 6篇文学创作）

## 执行逻辑

1. **生成入门内容**：为AI/科技圈新人生成6篇基础知识文档
2. **生成创作内容**：为AI文学创作新人生成6篇创作指南文档
3. **保存文件**：保存到 `data/萌新学习/` 目录，文件名格式为 `序号-标题.md`
4. **渲染到首页**：通过前端JavaScript自动读取 `data/萌新学习/` 目录下的文件并渲染到首页

## 错误处理

- **文件保存失败**：输出错误信息并尝试重试
- **目录不存在**：自动创建目录

## 使用方式

当用户请求执行萌新学习内容生成时：
1. 调用此skill
2. 执行内容生成
3. 生成并保存学习文档
4. 告知用户文档已生成完成

## 注意事项

- 内容面向完全不懂技术的萌新，语言要通俗易懂
- AI文学创作部分要包含实际案例和可操作的步骤
- 每篇文档要包含相关链接，方便读者进一步学习
- 文档格式要与现有的萌新学习文档保持一致
